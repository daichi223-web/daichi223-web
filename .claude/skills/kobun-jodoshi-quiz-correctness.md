# kobun-tan 助動詞クイズ 正誤ルール

ユーザー（高校国語教師）の指摘で確立した、作問・点検時に守るルール。
検出は `scripts/check-grammar-quiz.py`（係り結び）・`scripts/audit-grammar-all.py`（総合）、
手がかり追加は `scripts/apply-context-marks.py`。

## 1. つ・ぬ の「完了 / 強意（確述）」

- **接続は常に連用形**。だから **「連用形だから完了」は誤り**。連用形であること自体は
  *意味*（完了か強意か）の手がかりにならない。
  - ✗ 意味問題の説明「直前が連用形→完了」
  - ○ 「動作の完了・実現を表す→完了。下に推量〈む・べし〉が続けば強意」
  - ○ **識別問題**（完了ぬ vs 打消ず連体形ぬ）では「連用形接続→打消ずでなく完了の助動詞ぬ」は**正しい**手がかり（接続で正体を識別している）。意味の根拠と混同しないこと。
- **強意（確述）になるのは「未確定」のとき**＝下に **未来推量「む・べし」**（散り**ぬべし**＝きっと散るだろう）。
- **らむ（現在推量）・けむ（過去推量）が下接**するときは、動作はすでに実現した事態を
  推量しているので **完了のまま**でよい（「なり**ぬらむ**」＝なってしまっているだろう＝完了＋現在推量）。
- **確述（強意）は推量語が必須ではない**。旺文社『全訳古語辞典』は「**日も暮れぬ**」を確述の例とし、
  命令形「とく帰りたまひ**ね**」も確述。文末・命令でも成立する。詳細 [[kobun-nu-tsu-kanyu-kakujutsu]]。
- 判別の本質は **確定／未確定**。未確定＝強意、実現済み＝完了。

## 2. 終止形は「句切れ・文末」で決まる

- 終止形の手がかりは **句切れ／文末** であって、**後続の語ではない**。
  和歌の二句切れや「。」「会話文の『』」の直後の語に `《》` を引くのは**誤り**。
  - 実例：「慰めかね**つ**／更級や…」の《更級》、「追ひ払ひ**つ**。「**あはれ**…」の《あはれ》、
    文末「けり。」の次文の語── すべて誤マーク。削除し説明を「句切れ（文末）で終止形」に。
- ただし **終止形接続の語**（らむ・らし・べし・まじ・なり〈伝聞〉・めり・とも・引用の「と」・
  終助詞 な/かし/ぞ 等）が下接する場合は、その後続語が正しい手がかり。

## 3. 係り結びの手がかりは係助詞（遠隔でも）

- 連体形（ぞ・なむ・や・か）／已然形（こそ）は、結びの語の**前にある係助詞**が手がかり。
  後続語に振るのは誤り。遠く離れていても係助詞に `《》` を引く。[[kobun-grammar-quiz-kakari-check]]

## 4. てしがな・にしがな は願望の終助詞（連語）

- 「得**てしがな**」の「て」を「完了つ連用形」と問うのは不適切。**てしがな＝自己願望の終助詞**
  として扱う（同じ歌を joshi-shujoshi 側では正しく願望終助詞としている）。

## 5. レベルと未習事項

- **助動詞のL1（基礎）に、終助詞を正解にする問題を置かない**（助詞は未習）。
  例：願望の助動詞 ganbou のL1で「相手への願望＝終助詞なむ」を正解にしていた → 助動詞
  （まほし・たし）だけで完結する問題に差し替え。

## 6. 手がかり《》を引く/引かないの原則

- 引くのは **意味・活用を決める「語」が文中にある**場合のみ（係助詞・打消・呼応の「え」・
  反語副詞・心情動詞・尊敬語・場所体言＋なり・下接の推量 等）。広い文脈を見て遠隔でも引く。
- 決め手が **書かれていない主語**（推量＝三人称／意志＝一人称）や **地の文かどうか**（過去／詠嘆）
  の場合は、引く語が無いので**引かない**（本文全体を読んで自力で確定する設計）。目的は
  「文脈を広く見て意味を確定できるようにする」こと。[[kobun-grammar-level-calibration]]
