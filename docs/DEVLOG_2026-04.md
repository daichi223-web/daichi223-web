## 残課題・既知の問題

### 学習データの問題
- `kobun_q.jsonl.txt` 行530/538/540 が `sense: ""` で空 (dataParserで自動スキップ)
- `group: 43` が欠番 (データ抜け)
- `group 42: あまた + げに` / `group 270: かたみ + かたみに` が1群2lemma

### 未配線機能
- `srsEngine.ts` (updateSrsState / getDueWords 等) が App.tsx から呼ばれていない → srs_state テーブル空のまま
- `/api/aggregateCandidates` は手動トリガのみ (Vercel Cron未設定)

### セキュリティ懸念
- RLS ポリシーが `using (true)` で実質オープン (word_stats, srs_state)
- ADMIN_VIEW_TOKEN が localStorage → XSS 耐性なし
- クエリパラメータ `?token=...` がブラウザ履歴に残る

### 運用
- Obsidian Vault (kokugo-vault) 更新時は `py scripts/build-vocab-bundle.py && py scripts/extract-examples.py` を手動実行する必要あり
- CI化されていない
