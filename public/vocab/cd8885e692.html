かたじけなし

基本データ

品詞: 形容詞・ク活用 活用: かたじけなく／かたじけなく／かたじけなし／かたじけなき／かたじけなけれ／○ 語源: 「忝し（かたじけなし）」＝ 自分にはもったいない → 恐れ多い。

辞書的意味

番号 意味 典型的な例文 出典
❶ 恐れ多い、もったいない かたじけなき御心ばへ 源氏物語
❷ ありがたい、恐縮だ かたじけなく思ふ 大鏡

派生関係：

原義：忝し → 身に余って恐縮する
  ├→ ❶ 身分の高い方の恩恵が身に余る → 恐れ多い、もったいない（最重要義）
  └→ ❷ 恩恵に対する恐縮 → ありがたい、かたじけない

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶恐れ多い 天皇・高貴な人の恩恵を受ける場面。分不相応の感謝。 「御心ばへ」「御涙」「御こと」
❷ありがたい 恩恵に対して恐縮する場面。 「思ふ」「覚ゆ」
判別のコツ

身分差がある恩恵 → ❶ 恐れ多い 恩恵への感謝・恐縮 → ❷ ありがたい → ❶と❷は連続的で、「恐れ多くもありがたい」が多い。

用例（意味番号付き）

意味 用例 文脈解説
❶恐れ多い 「かたじけなき御心ばへのたぐひなきを頼みにてまじらひたまふ」（源氏物語） 恐れ多いお心遣いが比類ないのを頼みに宮仕えする→帝の特別な寵愛
❶もったいない 「御涙を流させ給ふぞかたじけなき」（平家物語） 御涙を流しなさるのは恐れ多い→法皇の涙
❷恐縮だ 「かたじけなく心苦しき御ことなめれば」（源氏物語） 恐れ多くお気の毒なことのようなので→恩恵への恐縮
❶恐れ多い 「いとかたじけなし」（大鏡） とても恐れ多い→高貴な方への畏怖

類義語・対義語

類義語 共通点 相違点
かしこし 「恐れ多い」で❶と近い 「かしこし」は畏怖全般、「かたじけなし」は恩恵への恐縮
ありがたし 「ありがたい」で❷と近い 「ありがたし」はめったにない、「かたじけなし」は身に余る
対義語 意味
おろかなり いいかげんだ、おろそかだ

入試での注意点

最重要

「かたじけなし」＝「恐れ多い・もったいない」。 現代語「かたじけない」とほぼ同義だが、古文では身分差の文脈が必須。

  • 源氏物語「桐壺」巻で帝の恩恵を表す場面が典型
  • 平家物語でも重要
  • 「かたじけなき御〜」のパターンが多い

覚え方のコツ

覚え方

「かたじけなし」＝ 「忝い（かたじけない）」＝ 身に余って恐れ多い → 時代劇で「かたじけない！」と言う場面そのもの。

イメージ: 殿様から直接褒美をもらって「恐れ多い…！」→ かたじけなし

ポイント: 恩恵＋恐縮のセット。源氏物語の帝の御心ばへが典型。

暗記用まとめ

一言で

「かたじけなし」＝恐れ多い・もったいない。身に余る恩恵への恐縮。

最重要: 「忝し」が本字。時代劇の「かたじけない！」のイメージ。身分差の文脈で使う。
