なかなか

基本データ

品詞: 副詞（形容動詞「なかなかなり」もあり） 語源: 「中＋中」＝中途半端 → かえって → むしろ。

辞書的意味

番号 意味 典型的な例文 出典
❶ かえって、むしろ なかなか長き夜 百人一首
❷ 中途半端に、なまじ なかなかなる心 源氏物語
❸ とても、かなり（＋否定で「容易に～ない」） なかなか言ふべきにあらず 徒然草

派生関係：

原義：中途半端なところ
  ├→ ❶ 中途半端なせいで → かえって逆の結果 → むしろ（最重要義）
  ├→ ❷ 中途半端だ → なまじっか
  └→ ❸ 並大抵ではない → とても → 否定と呼応して「容易に～ない」
      → 現代語:「なかなか」＝意外に（肯定）/ なかなか～ない（否定）
現代語との決定的な違い

現代語「なかなか」＝ 意外に良い / 容易に～ない 古語「なかなか」＝ かえって、むしろ（❶が最重要） 古文では❶「かえって」の意味が圧倒的に多い。

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶かえって 中途半端な行為がかえって悪い結果を招く場面。「～しないほうがましだ」の含意。 「なかなか＋形容詞/動詞」
❷中途半端 形容動詞「なかなかなり」の形。半端で不十分な様子。 「なり」「に」
❸容易に～ない 否定語と呼応する場面。「なかなか＋否定」で「とうてい～ない」。 「ず」「じ」「まじ」
判別のコツ

肯定文の「なかなか」→ ❶ かえって 「なかなかなり」→ ❷ 中途半端だ 「なかなか＋否定」→ ❸ 容易に～ない

用例（意味番号付き）

意味 用例 文脈解説
❶かえって 「なかなかに人づてならで言ふよしもがな」（百人一首・小野小町） かえって人伝えでなく直接言う方法があればなあ→中途半端な伝聞がかえってもどかしい
❶かえって 「逢ひ見てののちの心にくらぶれば昔はものを思はざりけり」の発想 逢ってかえって恋が深まる→中途半端に逢ったことが逆効果
❸容易に～ない 「なかなか言ふべきにあらず」（徒然草） とうてい言うべきことではない→否定と呼応

類義語・対義語

類義語 共通点 相違点
かへりて 「かえって」で❶と同義 ほぼ同義だが「なかなか」の方が使用頻度高い
なまじ 「中途半端に」で❷と近い 「なまじ」は否定的ニュアンスがやや強い

入試での注意点

最重要

「なかなか」を「なかなか良い」と現代語で訳すのは不正解。 肯定文の「なかなか」＝かえってを即答できるかが勝負。

  • 和歌で「なかなか」は❶「かえって」の用法が圧倒的
  • 「なかなかに」＝かえって、むしろ
  • 「なかなかなり」（形容動詞）＝中途半端だ（❷）

覚え方のコツ

覚え方

「中＋中」＝ 中途半端 → かえって裏目に出る → 中途半端なことをしたから「かえって」悪くなった

例え: 半端に告白したら「かえって」気まずくなった → 「なかなか＝かえって」のイメージ

ポイント: 肯定文で「なかなか」→「かえって」一択

暗記用まとめ

一言で

「なかなか」＝かえって、むしろ。中途半端ゆえに逆の結果を招く。

最重要: 古文「なかなか」の中心義は「かえって」。現代語の「なかなか良い」ではない。
