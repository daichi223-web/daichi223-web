さながら

基本データ

品詞: 副詞 語源: 「さ（其の）」＋「ながら」→ そのままで → 全部。 ※ ❶「そのまま・もとのまま」と❷「すべて・全部」の2義が重要。❷が古文特有で入試頻出。

辞書的意味

番号 意味 典型的な例文 出典
❶ そのまま、もとのまま さながら臥して 源氏物語
❷ すべて、全部、残らず さながら灰燼となりにき 方丈記

派生関係：

原義：そのままの状態で
  ├→ ❶ 変化なし → そのまま・もとのまま
  └→ ❷ 全体がそのまま → すべて・全部・残らず ★入試重要
※ 「女君はさながら臥して」＝ 女君はそのまま横になって（❶）★源氏物語
※ 「さながら捨つべきなり」＝ そのまま捨てるべきだ（❶）★徒然草
※ 「七珍万宝さながら灰燼となりにき」＝ あらゆる宝がすべて灰になった（❷）★方丈記
  → ❶は「変化なし」、❷は「例外なし」のニュアンス。
  → 現代語「さながら～のよう」（まるで）は後の用法。
注意

❷「すべて・全部」が古文特有で入試頻出。 現代語「さながら夢のよう」（まるで）とは使い方が違う。 古文では❶❷が中心。

判別基準と文脈解説

意味 どんな文脈で？ 判別のポイント
❶そのまま 状態が変わらない場面。 主語が一つ・変化なし
❷すべて 全体に及ぶ場面。 「灰燼」「滅ぶ」等＋全体対象
判別のコツ

変化なし・維持 → ❶ そのまま 全体・残らず → ❷ すべて

用例（意味番号付き）

意味 用例 文脈解説
❶そのまま 「帰り入りて探りたまへば、女君はさながら臥して」（源氏物語） 戻って手探りなさると、女君はそのまま横になっていて→廃院の怪
❶そのまま 「さながら捨つべきなり」（徒然草） そのまま捨てるべきだ→大事を思い立つ人
❷すべて 「七珍万宝さながら灰燼となりにき」（方丈記） あらゆる宝がすべて灰になってしまった→安元の大火★

類義語・対義語

類義語 共通点 相違点
そのまま ❶と同義 和語的表現
あまた 「全部」で❷と近い 「あまた」は数量（たくさん）
ことごとく 「すべて」で❷と同義 副詞的・堅い表現

入試での注意点

最重要

「さながら」＝ そのまま / すべて。 ❷「すべて」が古文特有で入試頻出！ 方丈記「七珍万宝さながら灰燼」が超有名。

  • 「さながら臥す」＝ そのまま横になる（❶）
  • 「さながら灰燼と」＝ すべて灰に（❷）★

覚え方のコツ

覚え方

「さながら」＝ 「さ（その）」＋「ながら」→ そのままながら → そのまま → ❷は「そのまま全部が」→「すべて」に転じたもの。

ポイント: 方丈記「さながら灰燼」＝ すべて灰に。「すべて」を忘れない。

暗記用まとめ

一言で

「さながら」＝そのまま・すべて。

最重要: ❷「すべて」が古文特有。方丈記「さながら灰燼」が最重要出典。
