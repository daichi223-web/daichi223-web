てひどし

基本データ

品詞: 形容詞・ク活用 活用: てひどく／てひどく／てひどし／てひどき／てひどけれ／○ 語源: 「手＋酷し（ひどし）」＝ 手ひどい → 容赦がない、ひどい。 ※ ファイル名は「てざひどし」だが、標準形は「てひどし」。

辞書的意味

番号 意味 典型的な例文 出典
❶ ひどい、容赦がない てひどき仕打ち —
❷ はなはだしい、尋常でない てひどき雨 —

派生関係：

原義：手＋酷し → 手荒で容赦がない
  ├→ ❶ 仕打ちが容赦ない → ひどい、手ひどい（最重要義）
  └→ ❷ 程度がはなはだしい → 尋常でない

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶容赦がない 人の仕打ち・態度がひどい場面。 「仕打ち」「言ふ」
❷はなはだしい 自然現象・事態の程度が激しい場面。 「雨」「風」
判別のコツ

人の行為 → ❶ ひどい、容赦がない 自然・事態 → ❷ はなはだしい → 現代語「手ひどい」とほぼ同義。

用例（意味番号付き）

意味 用例 文脈解説
❶ひどい 「てひどき仕打ちにて」 ひどい仕打ちで→容赦のなさ
❷はなはだしい 「てひどき雨」 激しい雨→程度のはなはだしさ

類義語・対義語

類義語 共通点 相違点
いみじ 「はなはだしい」で❷と近い 「いみじ」はより広義、「てひどし」は容赦のなさに焦点
むごし 「ひどい」で❶と近い 「むごし」は残酷、「てひどし」は手荒さ

入試での注意点

最重要

「てひどし」＝「ひどい・容赦がない」。 現代語「手ひどい」とほぼ同義。

覚え方のコツ

覚え方

「手＋ひどし」＝ 手荒で容赦がない → 手ひどい → 現代語「手ひどい（やられ方）」そのもの。

ポイント: 現代語と同じ感覚でOK。

暗記用まとめ

一言で

「てひどし」＝ひどい・容赦がない。手荒な扱い。

最重要: 現代語「手ひどい」と同義。「手＋酷し」が語源。
