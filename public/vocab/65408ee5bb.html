わづらふ

基本データ

品詞: 動詞・ハ行四段 活用: わづらは／わづらひ／わづらふ／わづらふ／わづらへ／わづらへ 漢字: 「煩ふ」「患ふ」 語源: 「わぶ（侘ぶ）」の派生語とも、「わづらはし（煩はし）」の動詞形とも言われる ＝ 心や身に負担がかかる → 思い悩む・苦労する・病気になる。 ※ 現代語「煩う・患う」にも残る。「思ひ煩ふ」「渡り煩ふ」など複合動詞も多い。

辞書的意味

番号 意味 典型的な例文 出典
❶ 思い悩む、心を痛める、困惑する 物のためにわづらはず 徒然草
❷ 苦労する、難儀する 悩みわづらふ 土佐日記
❸ 【補助動詞】〜するのに困る、〜しかねる 渡りわづらふ 更級日記
❹ 病気になる、患う 重くわづらふ —

派生関係：

原義：心身に負担がかかる
  ├→ ❶ 心の負担 → 思い悩む・困惑する（精神的）
  ├→ ❷ 身体・行動の負担 → 苦労する・難儀する（身体的）
  ├→ ❸ 補助動詞として → 〜しかねる・〜に困る（動詞連用形＋わづらふ）
  └→ ❹ 深刻な身体的負担 → 病気になる・患う
※ 現代語「患う（病気）」と「煩う（悩む）」に分岐して残存
注意

「わづらふ」は4義を持つ多義語。文脈で訳し分けが必要。 ・精神的なら❶「悩む」 ・身体的・行動的なら❷「苦労する」 ・動詞連用形に付くと❸補助動詞「〜に困る・〜しかねる」 ・明確に病気の場面なら❹「患う」

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶思い悩む 心の葛藤・心配の場面。 「物」「心」「思ひ」
❷苦労する 行動の困難・難儀の場面。 「悩む」「難儀」「ほど」
❸〜に困る（補助） 動詞連用形＋わづらふ。 「渡りわづらふ」「思ひわづらふ」
❹病気になる 病・床に伏す場面。 「重く」「悩ましく」「床」
判別のコツ

単独で精神的文脈 → ❶ 悩む 単独で身体的・行動的文脈 → ❷ 苦労する 動詞連用形＋わづらふ → ❸ 〜に困る（補助動詞） 病気の場面 → ❹ 患う 複合動詞「思ひ煩ふ」「渡り煩ふ」で❸が頻出。

用例（意味番号付き）

意味 用例 文脈解説
❶思い悩む 「寛大にして極まらざる時は、喜怒これにさはらずして、物のためにわづらはず」（徒然草） 心が広く大らかで狭く限ることがないときは、喜怒の感情が邪魔となることなく、物事のために思い悩まない→心の持ち方の教え
❷苦労する 「漕ぎ上るに、川の水干て、悩みわづらふ」（土佐日記） 都に向けて淀川を船で漕ぎ上るが、川の水が少なくなって、難儀し苦労する→船旅の苦労
❸〜に困る 「勢多の橋みな崩れて、渡りわづらふ」（更級日記） 勢多の橋がみな崩れて、渡るのに困る→旅路の障害
❹患う 「父君の、重くわづらひ給へば」（源氏物語風） 父君がひどく患いなさるので→病床の親

類義語 共通点 相違点

入試での注意点

最重要

「わづらふ」＝❶思い悩む／❷苦労する／❸〜に困る（補助動詞）／❹患う。 動詞連用形＋わづらふ は補助動詞「〜しかねる・〜に困る」。

  • 「渡りわづらふ」「思ひわづらふ」は補助動詞用法（❸）
  • 徒然草で❶「悩む」頻出
  • 土佐日記・更級日記で❷❸の移動の難儀が多い
  • ❹「病気」は現代語「患う」と同じ感覚

覚え方のコツ

覚え方

「わづらふ」＝ 心身に負担 → 悩む・苦労する・困る・患う → 心に負担：❶ 悩む → 身に負担：❷ 苦労する → 動作に負担（補助動詞）：❸ 〜しかねる → 病気になる：❹ 患う

イメージ: 負担を感じている状態の全てを「わづらふ」で表現 → 現代語「煩う・患う」が分業しているのを古文では一語で表す

ポイント: 動詞連用形＋わづらふ は「〜に困る」（補助動詞）。必ず押さえる。
