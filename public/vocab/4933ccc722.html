したたむ

基本データ

品詞: 動詞・マ行下二段活用 活用: したため／したため／したたむ／したたむる／したたむれ／したためよ 漢字: 「認む／整む」 語源: 「確か（したたか）」と同根で、「しっかり整える・きちんと処理する」が原義。そこから「処理する／用意する／取り締まる／書きしたためる」など多方面に展開。

辞書的意味

番号 意味 典型的な例文 出典
❶ 処理する、てきぱきと片付ける 物ひきしたため 徒然草
❷ 用意する、準備する、整える 寝所などはしたためて 今昔物語集
❸ 取り締まる、監督する、統率する えしたためあへず 源氏物語
❹ 書きしるす、したためる（文章を書く） 手紙をしたたむ 類例

派生関係：

原義：確か（しっかり）に整える
  ├→ ❶ 物事を確かに処理する → 処理する・片付ける
  ├→ ❷ 備えを確かに整える → 用意する・準備する
  ├→ ❸ 乱れを確かにおさえる → 取り締まる・統率する
  └→ ❹ 文字を確かに書き付ける → したためる（書く）
※ 現代語「したためる（書き記す）」は❹が残存。
  ❶❷❸は古文特有の意味で、受験必須。
注意（多義語）

「したたむ」は4義ある多義語。文脈で❶❷❸❹を判断する。 現代語の「したためる（書く）」だけで訳すと大きく外す可能性がある。 源氏物語では❸「取り締まる」、徒然草では❶「処理する」、今昔物語集では❷「用意する」が頻出。 下二段活用（「したためず」「したためて」「したたむる」）。

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶処理する 仕事・事務をてきぱき片付ける場面。 「物」「ひき」「ちりぢり」
❷用意する 旅・防備・寝所などの準備の場面。 「寝所」「破籠」「食ひ物」
❸取り締まる 集団の乱れを統制する場面。 「若き者」「酔ひすぎ」「騒ぐ」
❹書きしるす 文章・手紙を書き付ける場面。 「手紙」「文」「詩」
判別のコツ

事務処理・片付け → ❶「処理する」（徒然草） 準備・用意 → ❷「用意する」（今昔物語集） 統制・監督 → ❸「取り締まる」（源氏物語） 手紙・文章 → ❹「書く」（現代語と同じ） 文脈の対象物で❶〜❹を決める。

用例（意味番号付き）

意味 用例 文脈解説
❶処理する 「果ての日は、いと情けなう、互ひに言ふこともなく、我かしこげに物ひきしたため、ちりぢりに行き別れぬ」（徒然草） 死後の供養の最後の日は、実に人情味なく、互いに口をきくこともなく、われがちに物をてきぱき処理し、散り散りに離れ去ってしまう→兼好の無常観
❷用意する 「これを思ふに、女なりともなほ寝所などはしたためてあるべきなり」（今昔物語集） これを思うと、女であってもやはり寝所などは（危険に備えて）用意しているべきである
❸取り締まる 「いづ方にも、若き者ども酔ひすぎたち騒ぎたるほどのことは、えしたためあへず」（源氏物語） どちらの方でも、若者たちが泥酔し騒いでいる間の振る舞いは、最後まで取り締まることができない
❹書く 「手紙をしたたむ」（現代語にも残る用法） 手紙を書きしるす

類義語・対義語

類義語 共通点 相違点
まうく（設く） 「用意する」で❷と近い 「まうく」は前もって準備、「したたむ」はきちんと整えて用意
おさむ（治む） 「取り締まる」で❸と近い 「おさむ」は広く治める、「したたむ」は乱れを整える
ものす 「処理する」で❶と近い 「ものす」はぼかし語、「したたむ」はしっかり処理
対義語 意味
おこたる 怠る、いい加減にする
おろそかなり 粗略だ

入試での注意点

最重要

「したたむ」＝4義（処理する／用意する／取り締まる／書く）。 現代語「したためる（書く）」だけで訳すと誤答。古文では❶❷❸が中心。 下二段活用（活用識別問題でも問われる）。

  • 徒然草「物ひきしたため」＝物をてきぱき処理する（❶）
  • 今昔物語集「寝所をしたためて」＝寝所を用意して（❷）
  • 源氏物語「えしたためあへず」＝取り締まりきれない（❸）
  • 奥の細道「こまやかにしたためさせ」＝心をこめて用意させ（❷）

覚え方のコツ

覚え方

「したたむ」＝ 確（したた）か＋む ＝ しっかり整える - 物事をしっかり整える → ❶ 処理する - 備えをしっかり整える → ❷ 用意する - 乱れをしっかり整える → ❸ 取り締まる - 文字をしっかり書く → ❹ したためる

イメージ: 徒然草「物ひきしたため」＝葬儀後にてきぱきと片付ける → 今昔「寝所をしたためて」＝寝所を用意する → 源氏「えしたためあへず」＝若者を取り締まりきれない

ポイント: 4義全部を押さえる。現代語の「したためる（書く）」だけでは不十分。

暗記用まとめ

一言で

「したたむ」＝処理する／用意する／取り締まる／書く。

最重要: 4義の多義語。下二段活用。徒然草「物ひきしたため＝処理する」、今昔「寝所をしたためて＝用意する」、源氏「えしたためあへず＝取り締まりきれない」が頻出3義。「書く」は現代語の名残。
