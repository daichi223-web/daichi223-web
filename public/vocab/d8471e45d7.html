このファイルは「助動詞判別」カテゴリの全エントリを結合したものです。（18語）

き・けりの判別

基本データ

き: 特殊型活用（せ／○／き／し／しか／○）。連用形に接続。 けり: ラ変型活用（けら／○／けり／ける／けれ／○）。連用形に接続。 共通: どちらも「過去」を表す。

意味の違い

助動詞 意味 ニュアンス 典型場面
き 直接体験の過去 自分が直接経験・目撃した 日記・回想「私は～した」
けり 間接体験の過去（伝聞過去） 人から聞いた・書物で知った 物語の地の文「～したそうだ」
けり 詠嘆（気づき） 「～だったのだなあ」 和歌・感動の場面

判別フローチャート

Q1: 和歌の中で使われているか？
  → Yes → けり = ❸詠嘆「～だったのだなあ」の可能性が高い
  → No → Q2へ

Q2: 語り手（筆者）が自分で体験した事柄か？
  → Yes → き（直接体験の過去）
  → No → けり（伝聞・物語の過去）

Q3: 「～にけり」「今ぞ知りける」など、気づき・感動の文脈か？
  → Yes → けり = 詠嘆

判別の詳細

助動詞 典型的な文脈 例文
き 日記・随筆で筆者が体験を語る 「いとをかしかりき」（枕草子）→ 私はとても趣深かった
けり 物語の地の文（語り手が伝える） 「昔、男ありけり」（伊勢物語）→ 昔、男がいたそうだ
けり（詠嘆） 和歌・発見の場面 「春は来にけり」→ 春が来たのだなあ
注意

「けり」は物語冒頭の常套句。「昔、～けり」は「～だったそうだ」。 和歌の「けり」は詠嘆が基本。「～けり」「～ける」→「～だったのだなあ」。

用例

種別 用例 解説
き（直接過去） 「春はあけぼの。やうやう白くなりゆく山ぎは、少しあかりて、紫だちたる雲の細くたなびきたる」→ 清少納言の実体験 筆者の直接体験
けり（伝聞過去） 「昔、男ありけり」（伊勢物語） 物語の語り出し→伝聞
けり（詠嘆） 「久方の光のどけき春の日にしづ心なく花の散るらむ」の類で「散りけるかな」 気づき・感動
けり（詠嘆） 「世の中にたえて桜のなかりせば春の心はのどけからまし」→ 「けり」ではないが、「き」の未然形「せ」の例 反実仮想の「せ…まし」

紛らわしいパターン

パターン 判別法
「し」（「き」の連体形） vs 「し」（副助詞・強意） 体言を修飾していれば「き」の連体形。「見し人」＝「かつて見た人」
「しか」（「き」の已然形） vs 「しか」（「然か」副詞） 接続助詞「ば」「ど」が後続すれば「き」の已然形
「せ」（「き」の未然形） vs 「せ」（「す」使役の未然形） 「せば…まし」の形なら「き」の未然形（反実仮想）

入試での注意点

最重要
  1. き ＝ 直接体験、けり ＝ 伝聞/詠嘆
  2. 和歌の「けり」はほぼ詠嘆
  3. 「し」が連体形か副助詞かの判別が頻出
  4. 「せば…まし」の「せ」は「き」の未然形

暗記用まとめ

一言で

き＝自分で見た・した（直接過去）、けり＝聞いた話（伝聞）/気づき（詠嘆） 物語冒頭「～けり」＝伝聞。和歌「～けり」＝詠嘆。日記「～き」＝体験。

けむの判別

基本データ

けむ: 四段型活用（○／○／けむ／けむ／けめ／○）。連用形に接続。 意味: 過去推量・原因推量・婉曲の3つ。 ポイント: 「む」の過去版。「む」は未来、「らむ」は現在、「けむ」は過去。

3つの意味

番号 意味 訳し方
❶ 過去推量 ～ただろう
❷ 原因推量 どうして～ただろうか
❸ 婉曲 ～たような

判別法

「らむ」と同じフローチャート

疑問詞あり→原因推量。連体形→婉曲。それ以外→過去推量。

用例

意味 用例 解説
❶過去推量 「昔ここに住みけむ人」 「昔ここに住んでいただろう人」
❷原因推量 「いかに思ひけむ」 「どう思っただろうか」
❸婉曲 「見けむ人ぞ知る」 「見たような人が知っている」

暗記用まとめ

一言で

けむ＝過去推量/原因推量/婉曲。「む」の過去版。「らむ」と判別法は同じ。

じの判別

基本データ

じ: 無変化型。未然形に接続。 意味: 「む」の打消版。打消推量・打消意志の2つ。

2つの意味

番号 意味 訳し方 「む」との対応
❶ 打消推量 ～ないだろう 推量「む」の打消
❷ 打消意志 ～するまい、～しないつもりだ 意志「む」の打消

判別法

「む」と同じ

一人称→打消意志、三人称→打消推量。

用例

意味 用例 解説
❶打消推量 「雨は降らじ」 「雨は降らないだろう」
❷打消意志 「二度と行かじ」 「二度と行くまい」
❷打消意志 「人に見えじとて」 「人に見られまいとして」

暗記用まとめ

一言で

じ＝打消推量/打消意志。「む」の打消版。 一人称→打消意志「～するまい」、三人称→打消推量「～ないだろう」。

す・さす・しむの判別

基本データ

す: 下二段型。四段・ナ変・ラ変の未然形に接続。 さす: 下二段型。上記以外の未然形に接続。 しむ: 下二段型。未然形に接続（漢文訓読で多用）。 意味: 使役・尊敬の2つ。

2つの意味

番号 意味 訳し方 例
❶ 使役 ～させる 人を走らす
❷ 尊敬 ～なさる 帝、詠ませ給ふ

判別フローチャート

Q1: 「せ/させ＋給ふ」の形か？
  → Yes → ❷尊敬（尊敬の二重敬語）
  → No → Q2へ

Q2: 主語が高貴で、「～を…す/さす」の使役対象がないか？
  → 使役対象なし＋高貴な主語 → ❷尊敬
  → 使役対象あり（「～に」「～を」で示される）→ ❶使役

判別の詳細

意味 手がかり
❶使役 動作をさせられる人物が「に」「を」で示される。「人に書かす」「子を走らす」
❷尊敬 「す/さす」＋「給ふ」で尊敬の二重敬語。また、主語が高貴で使役対象がない場合。
最重要ポイント

「せ給ふ」「させ給ふ」は尊敬の二重敬語。使役＋尊敬ではなく、全体で一つの尊敬表現。 書き下し文で「しむ」が出たら、直前は未然形（読ましむ、行かしむ）。

用例

意味 用例 解説
❶使役 「人に門を開けさせて」 「人に」が使役の対象→使役
❶使役 「郎等を走らせて」 「郎等を」が使役の対象→使役
❷尊敬 「帝、歌を詠ませ給ふ」 「せ給ふ」→尊敬の二重敬語
❷尊敬 「宮、御覧ぜさせ給ふ」 「させ給ふ」→尊敬の二重敬語

入試での注意点

最重要
  1. 「せ給ふ」「させ給ふ」＝ 尊敬の二重敬語（使役＋尊敬ではない）
  2. 使役は動作の受け手が「に」「を」で明示される
  3. 漢文の「しむ」→ 直前は未然形

暗記用まとめ

一言で

す・さす・しむ = 使役/尊敬 「せ給ふ」「させ給ふ」→ 尊敬の二重敬語。使役は動作の受け手が「に」「を」で示される。

たり・りの判別

基本データ

たり: ラ変型活用（たら／たり／たり／たる／たれ／たれ）。連用形に接続。 り: ラ変型活用（ら／り／り／る／れ／れ）。サ変の未然形・四段の已然形に接続。 意味: 完了・存続の2つ（共通）。 「り」の接続の語呂: 「さみしい（サ未四已）」

2つの意味

番号 意味 訳し方
❶ 完了 ～た、～てしまった
❷ 存続 ～ている、～てある

完了と存続の判別

Q: 動作・状態が続いているか？
  → Yes → ❷存続「～ている」
  → No（瞬間的な完了）→ ❶完了「～た」
意味 典型 例
❶完了 一回的・瞬間的な動作 「書きたり」（書いた）
❷存続 状態が継続中 「門開けたり」（門が開けてある＝開いた状態が続く）

「り」の接続：「さみしい」

最重要語呂合わせ

「さみしい」＝ サ変の未然形・四段の已然形 - サ変未然形：「せり」（す→せ＋り） - 四段已然形：「咲けり」（咲く→咲け＋り）

接続 例
サ変未然形＋り 「御覧ぜり」（御覧になった/なっている）
四段已然形＋り 「花咲けり」（花が咲いた/咲いている）

用例

意味 用例 解説
❶完了（たり） 「既に書きたり」 「既に書いた」→完了
❷存続（たり） 「門開けたり」 「門が開けてある」→存続
❶完了（り） 「花散れり」 「花が散った」→完了
❷存続（り） 「花咲けり」 「花が咲いている」→存続
存続（り） 「雪降れり」 「雪が降っている」→存続

入試での注意点

最重要
  1. 「り」の接続＝「さみしい」（サ未四已）
  2. 完了か存続かは文脈で判断（状態の継続→存続）
  3. 「たり」と断定の「たり」を混同しない（断定は体言に接続）

暗記用まとめ

一言で

たり・り＝完了「～た」/存続「～ている」 「り」の接続は「さみしい」（サ未四已）。状態が続いていれば存続。

つ・ぬの判別

基本データ

つ: 下二段型活用（て／て／つ／つる／つれ／てよ）。連用形に接続。 ぬ: ナ変型活用（な／に／ぬ／ぬる／ぬれ／ね）。連用形に接続。 意味: 完了・強意の2つ。

意味

番号 意味 訳し方
❶ 完了 ～た、～てしまった
❷ 強意（確述） きっと～、必ず～

「つ」と「ぬ」の使い分け

つ ぬ
ニュアンス 意志的・人為的 自然的・無意志的
典型動作 「書きつ」「取りつ」「言ひつ」 「散りぬ」「暮れぬ」「死にぬ」
覚え方 人がやること＝つ 自然に起こること＝ぬ

強意の判別

「つ」「ぬ」が強意になるのは、推量の助動詞の直前のみ。

パターン 意味
て＋む → 「てむ」 きっと～だろう、～してしまおう
な＋む → 「なむ」 きっと～だろう
に＋けり → 「にけり」 ～してしまった（完了）
て＋む＋ず → 「てじ」 きっと～ないだろう
ぬ＋べし → 「ぬべし」 きっと～にちがいない、今にも～しそうだ
つ＋べし → 「つべし」 きっと～にちがいない
最重要ルール

「つ/ぬ」＋推量系助動詞（む・べし・らむ・けむ・まし・じ）＝ 強意 それ以外は完了。

判別フローチャート

Q1: 「つ/ぬ」の直後に推量系助動詞（む・べし・らむ・まし・じ等）があるか？
  → Yes → ❷強意「きっと～」
  → No → ❶完了「～た、～してしまった」

用例

意味 用例 解説
❶完了（つ） 「書きつ」 書いた・書いてしまった（意志的完了）
❶完了（ぬ） 「花散りぬ」 花が散った・散ってしまった（自然的完了）
❷強意 「死にてむとす」 きっと死んでしまおうとする（て＋む→強意）
❷強意 「風立ちぬべし」 きっと風が立つにちがいない（ぬ＋べし→強意）
❷強意 「泣きぬべし」 今にも泣いてしまいそうだ

紛らわしい「ぬ」の判別

→ 詳しくは ぬ・ねの判別 を参照。

形 正体 判別法
「散りぬ」 完了「ぬ」終止形 連用形＋ぬ → 完了
「散らぬ」 打消「ず」連体形 未然形＋ぬ → 打消

入試での注意点

最重要
  1. つ＝意志的、ぬ＝自然的（ニュアンスの違い）
  2. 「つ/ぬ」＋推量系＝強意（それ以外は完了）
  3. 「ぬ」と打消「ず」の連体形「ぬ」の区別が超頻出

暗記用まとめ

一言で

つ＝意志的完了、ぬ＝自然的完了。推量の直前なら強意。 「ぬべし」「てむ」「なむ」→ 強意。それ以外→ 完了。

「なむ」の判別

基本データ

「なむ」は古文で4通りの解釈が可能。入試超頻出の判別問題。

4つの「なむ」

番号 正体 接続 意味
❶ 強意「ぬ」の未然形「な」＋推量「む」 連用形＋なむ きっと～だろう
❷ 終助詞「なむ」（願望） 未然形＋なむ ～してほしい
❸ 係助詞「なむ」（強調） 体言・連用形等＋なむ ～は（強調）→結びは連体形
❹ ナ変動詞「死ぬ」「往ぬ」の未然形＋む 「死な＋む」「往な＋む」 死のう、行こう

判別フローチャート

Q1: 直前は何形か？
  → 連用形 → ❶強意＋推量「きっと～だろう」
  → 未然形 → ❷終助詞（願望）「～してほしい」 or ❹ナ変＋む
  → 体言・副詞等 → ❸係助詞（強調）

Q2: 未然形の場合、ナ変動詞（死ぬ・往ぬ）の未然形か？
  → Yes → ❹「死のう」「行こう」
  → No → ❷願望「～してほしい」

判別の詳細

種類 接続例 例文
❶強意＋推量 「散りなむ」（散り＝連用形） 「花はきっと散るだろう」
❷終助詞（願望） 「来なむ」？→ 注意！「来」がカ変未然形「こ」なら「こなむ」で願望 「来てほしい」
❸係助詞 「花なむ咲きたる」 「花が（こそ）咲いている」→ 結び連体形
❹ナ変＋む 「死なむ」（死ぬの未然形「死な」＋む） 「死のう」

用例

種別 用例 解説
❶強意＋推量 「風吹きなむ」 吹き（連用形）＋な＋む→「きっと風が吹くだろう」
❷願望 「花咲かなむ」 咲か（未然形）＋なむ→「花が咲いてほしい」
❸係助詞 「この花なむいとをかしき」 体言＋なむ→係助詞。結び「をかしき」は連体形。
❹ナ変＋む 「ここにて死なむ」 死な（ナ変未然形）＋む→「ここで死のう」

入試での注意点

最重要
  1. 接続で判別：連用形＝強意＋推量、未然形＝願望、体言＝係助詞
  2. 「花咲かなむ」（未然形＋なむ）＝願望「咲いてほしい」
  3. 「花散りなむ」（連用形＋なむ）＝強意＋推量「きっと散るだろう」
  4. 係助詞「なむ」は結びが連体形になる（係り結び）

暗記用まとめ

一言で

連用形＋なむ＝強意推量、未然形＋なむ＝願望、体言＋なむ＝係助詞 接続する活用形で判別。入試超頻出。

なりの判別（断定 vs 伝聞推定）

基本データ

断定の「なり」: ナリ活用型（なら／なり・に／なり／なる／なれ／なれ）。体言・連体形に接続。 伝聞推定の「なり」: ラ変型（なら／なり／なり／なる／なれ／○）。終止形に接続（ラ変は連体形に接続）。 ※ 活用形だけでは区別しにくい。接続で判別するのが鉄則。

2つの「なり」

断定の「なり」 伝聞推定の「なり」
意味 ～だ、～である ～らしい、～と聞いている
接続 体言・連体形 終止形（ラ変は連体形）
成り立ち 「に＋あり」 「音（ね）＋あり」

判別フローチャート

Q1: 「なり」の直前の語を確認。
  → 体言（名詞）の直後 → 断定「～である」
  → 連体形の直後 → Q2へ
  → 終止形の直後 → 伝聞推定「～らしい」

Q2: 直前がラ変動詞（あり・をり・はべり・いまそかり）の連体形か？
  → Yes → 伝聞推定の可能性あり（ラ変は連体形接続のため）
  → No → 断定
最重要ポイント

終止形と連体形が同形の活用（四段・ラ変等）では判別が難しい。 ラ変動詞の後の「なり」は要注意。 文脈で断定か伝聞推定か判断。

判別の実践

直前の語 接続 判定 例
体言 体言＋なり 断定 「男なり」（男である）
四段終止形 終止形＋なり 伝聞推定 「行くなり」（行くらしい）
四段連体形 連体形＋なり 断定 「行くなる人」→ ここの「なる」は連体形。「行くである人」→断定
下二段終止形 「見ゆ＋なり」 伝聞推定 「見ゆなり」（見えるようだ）
ラ変連体形 「ある＋なり」 要注意 文脈で断定か伝聞推定か判断

特別な判別パターン

パターン 判定 理由
「に」の形 断定の「なり」の連用形 伝聞推定に「に」の活用形はない
「にて」「にや」「にこそ」 断定 断定の連用形「に」＋助詞
「なりけり」 伝聞推定 断定なら「にけり」になるはず
「ならば」 断定 未然形は両者同形だが、仮定文脈は断定が多い
超重要テクニック

「なり」→「に」に置き換えられるか？ → 置き換え可能 ＝ 断定（「男なり」→「男に」OK） → 置き換え不可能 ＝ 伝聞推定（「行くなり」→「行くに」×）

用例

種別 用例 解説
断定 「これは竹取の翁と言ふ者なりけり」 「者」（体言）の直後→断定「者である」
断定 「男もすなる日記」 「す」＋「なる」→「す」はサ変終止形＝断定… ではなく「する」連体形→断定。※ 実は終止形「す」→伝聞推定「するという」が正解。超頻出問題！
伝聞推定 「男もすなる日記と言ふもの」 「す」（サ変終止形）＋「なる」→伝聞推定「男もするという日記」
伝聞推定 「笛吹くなり」 「吹く」（四段終止形）＋「なり」→「笛を吹いているようだ」（音が聞こえる）
断定 「花の色は移りにけりな」 「に」＝完了「ぬ」の連用形（断定の「なり」ではない）→ 別の「に」

入試超頻出：「男もすなる日記」

最重要

「男もすなる日記」の「なる」＝ 伝聞推定の連体形。 理由：「す」はサ変動詞の終止形。終止形に接続→伝聞推定。 訳：「男も書くという日記」

暗記用まとめ

一言で

断定＝体言/連体形に接続、伝聞推定＝終止形に接続。 「に」に置き換え可能→断定。「なりけり」→伝聞推定。「男もすなる」→伝聞推定。

「に」の判別

基本データ

「に」は古文で最も判別が難しい語の一つ。以下の可能性がある。 1. 断定の助動詞「なり」の連用形 2. 完了の助動詞「ぬ」の連用形 3. 格助詞「に」（場所・時・対象・原因等） 4. 接続助詞「に」（～ので、～のに） 5. 形容動詞の活用語尾（「静かに」等）

判別フローチャート

Q1: 直前は体言（名詞）か？
  → Yes → 断定「なり」の連用形 or 格助詞「に」 → Q2へ
  → No → Q3へ

Q2: 「にて」「にや」「にこそ」「にあり」の形か？
  → Yes → 断定「なり」の連用形
  → No → 格助詞「に」

Q3: 直前は連用形（動詞）か？
  → Yes → 完了「ぬ」の連用形 or 格助詞「に」
  → 「に＋けり」「に＋き」「に＋たり」→ 完了「ぬ」の連用形
  → その他 → 格助詞

Q4: 直前は形容動詞の語幹か？
  → Yes → 形容動詞の連用形活用語尾

各「に」の詳細

1. 断定「なり」の連用形

手がかり 例
体言＋「にて」「にや」「にこそ」「にあり」 「男にてあれば」（男であるので）
体言＋「に」＋「あり」系動詞 「僧にこそありけれ」（僧であったのだ）

2. 完了「ぬ」の連用形

手がかり 例
連用形＋「に」＋過去系助動詞（けり・き・たり） 「散りにけり」（散ってしまった）
連用形＋「に」＋「あり」 「行きにたり」（行ってしまっている）

3. 格助詞「に」

用法 例
場所 「山に住む」
時 「夜に鳴く」
対象 「人に言ふ」
原因 「風に散る」

4. 接続助詞「に」

手がかり 例
連体形＋「に」（文の切れ目） 「雨降るに、出でず」（雨が降るので、出ない）

用例で判別練習

用例 「に」の正体 判別理由
「男にてありければ」 断定「なり」の連用形 体言＋「にて」→断定
「花散りにけり」 完了「ぬ」の連用形 連用形＋「に」＋「けり」→完了
「山に登る」 格助詞 場所を示す
「雨降るに」 接続助詞 連体形＋「に」→接続助詞「～ので」
「静かに語る」 形容動詞活用語尾 ナリ活用の連用形

入試での注意点

最重要
  1. 「にけり」「にき」→ 完了「ぬ」の連用形
  2. 「にて」「にや」「にこそ」→ 断定「なり」の連用形
  3. 体言＋「に」＋動詞→ 格助詞が多い
  4. 連体形＋「に」（文の切れ目）→ 接続助詞

暗記用まとめ

一言で

「に」＝断定/完了/格助詞/接続助詞/形容動詞語尾 「にけり」→完了、「にて・にや」→断定、体言＋に＋動詞→格助詞。

ぬ・ねの判別（完了 vs 打消）

基本データ

完了の「ぬ」: ナ変型（な／に／ぬ／ぬる／ぬれ／ね）。連用形に接続。 打消「ず」の連体形「ぬ」: （ず／ず／ず／ぬ／ね／○）。未然形に接続。 完了の「ね」: 完了「ぬ」の命令形。 打消の「ね」: 打消「ず」の已然形。

判別法：接続で100%判別可能

完了「ぬ」 打消「ず」の「ぬ」
接続 連用形＋ぬ 未然形＋ぬ
例 散りぬ（散りは連用形） 散らぬ（散らは未然形）
最重要ルール

直前の動詞が連用形なら完了、未然形なら打消。これだけ。

活用形の見分け方（四段動詞の場合）

動詞 未然形（打消に接続） 連用形（完了に接続）
行く 行か＋ぬ（打消） 行き＋ぬ（完了）
散る 散ら＋ぬ（打消） 散り＋ぬ（完了）
咲く 咲か＋ぬ（打消） 咲き＋ぬ（完了）
思ふ 思は＋ぬ（打消） 思ひ＋ぬ（完了）

「ね」の判別

完了「ぬ」の命令形「ね」 打消「ず」の已然形「ね」
接続 連用形＋ね 未然形＋ね
後続 文末・命令 「ば」「ど」が後続
例 「早く行きね」（早く行ってしまえ） 「行かねば」（行かないので）

判別フローチャート

Q1: 直前の動詞は連用形か未然形か？
  → 連用形（四段なら「い段」）→ 完了「ぬ」
  → 未然形（四段なら「あ段」）→ 打消「ず」の連体形

Q2:「ね」の場合、後ろに「ば」「ど」があるか？
  → Yes → 打消「ず」の已然形「ね」
  → No（文末）→ 完了「ぬ」の命令形「ね」

用例

種別 用例 解説
完了 「花散りぬ」 散り（連用形）＋ぬ →「花が散ってしまった」
打消 「花散らぬ庭」 散ら（未然形）＋ぬ →「花が散らない庭」
完了命令 「疾く行きね」 行き（連用形）＋ね →「早く行ってしまえ」
打消已然 「行かねば」 行か（未然形）＋ね＋ば →「行かないので」
完了 「春は来ぬ」 来（「く」の連用形「き」）＋ぬ → 「春が来た」 ※「来ない」ではない！

超頻出：「春は来ぬ」

最重要

「春は来ぬ」＝「春が来た」（完了）。「春が来ない」ではない！ 「来（く）」はカ変活用：こ／き／く／くる／くれ／こよ 「来（き）」は連用形→完了「ぬ」に接続。

入試での注意点

最重要
  1. 連用形＋ぬ＝完了、未然形＋ぬ＝打消
  2. 「春は来ぬ」＝完了（超頻出誤答）
  3. 「ね」の後に「ば」「ど」→打消、文末→完了命令
  4. 上二段・下二段動詞は未然形と連用形が同形のことがあるので要注意

暗記用まとめ

一言で

連用形＋ぬ＝完了「～した」、未然形＋ぬ＝打消「～ない」 接続する活用形で100%判別可能。「春は来ぬ」＝「春が来た」。

べしの判別

基本データ

べし: 形容詞ク活用型（べく／べく／べし／べき／べけれ／○）。終止形に接続（ラ変は連体形に接続）。 意味: 推量・意志・可能・当然・命令・適当の6つ。 語呂合わせ: 「す・い・か・と・め・て」（推量・意志・可能・当然・命令・適当）

6つの意味

番号 意味 訳し方 語呂
❶ 推量 ～だろう、～にちがいない すいか
❷ 意志 ～しよう、～するつもりだ すいか
❸ 可能 ～できる すいか
❹ 当然・義務 ～すべきだ、～なければならない とめて
❺ 命令 ～せよ とめて
❻ 適当 ～するのがよい とめて

判別フローチャート

Q1: 主語は誰か？
  → 一人称 → ❷意志「～しよう」が最有力
  → 二人称 → ❹当然「～すべきだ」または❺命令「～せよ」
  → 三人称 → ❶推量「～だろう」

Q2: 打消・疑問・反語の文脈か？
  → 「～べからず」→ ❸可能「～できない」or ❹当然「～すべきでない」
  → 「～べきにあらず」→ ❹当然の打消

Q3: 「～するのがよい」で自然か？
  → Yes → ❻適当

Q4: 社会的義務・道理の文脈か？
  → Yes → ❹当然「～すべきだ」

判別の詳細

意味 手がかり 例文
❶推量 三人称主語。確信度が高い推量。 「雨降るべし」（雨が降るにちがいない）
❷意志 一人称主語。「～しよう」。 「我行くべし」（私は行くつもりだ）
❸可能 打消と共起が多い。「～できる」。 「言ふべくもあらず」（言うこともできない）
❹当然 道理・義務・規範の文脈。 「人は誠あるべし」（人は誠実であるべきだ）
❺命令 命令・指示の文脈。二人称。 「早く参るべし」（早く参上せよ）
❻適当 「～するのがよい」。助言。 「静かに暮らすべし」（静かに暮らすのがよい）

用例

意味 用例 解説
❶推量 「日本にも名の聞こゆべき人」 「名前が知れ渡るにちがいない人」
❷意志 「いかにもして、行くべしと思ふ」 「何としても行こうと思う」
❸可能 「言ふべき方もなし」 「言うこともできるすべもない」
❹当然 「学問は日々怠るべからず」 「学問は日々怠るべきではない」
❺命令 「疾く参るべし」 「早く参上せよ」
❻適当 「よき友に交はるべし」 「よい友と交際するのがよい」

「む」との違い

む べし
推量の確信度 低い（～だろう） 高い（～にちがいない）
意味の数 5つ 6つ（可能・当然・命令・適当が追加）
判別の難易度 人称で判別しやすい 6択で難しい

入試での注意点

最重要
  1. 語呂「すいかとめて」で6つの意味を暗記
  2. 人称で絞る：一人称＝意志、二人称＝当然/命令、三人称＝推量
  3. 打消文の「べからず」は可能か当然で迷う→文脈で判断
  4. 「べし」は確信度の高い「む」と理解する

暗記用まとめ

一言で

べし ＝ す・い・か・と・め・て（推量・意志・可能・当然・命令・適当） 人称で絞り、文脈で確定。「む」より確信度が高い。

ましの判別

基本データ

まし: 特殊型活用（ませ・ましか／○／まし／まし／ましか／○）。未然形に接続。 意味: 反実仮想・ためらいの意志の2つ。

2つの意味

番号 意味 訳し方 典型パターン
❶ 反実仮想 （もし）～だったら、～だろうに 「ませば…まし」「せば…まし」
❷ ためらいの意志 ～しようかしら 疑問の文脈

判別フローチャート

Q1: 「ませば…まし」「せば…まし」の形か？
  → Yes → ❶反実仮想「もし～なら、～だろうに」
  → No → Q2へ

Q2: 疑問・迷いの文脈か？（「いかにせまし」等）
  → Yes → ❷ためらいの意志「～しようかしら」
  → No → ❶反実仮想

反実仮想の公式

超重要公式

「未然形＋ば…まし」＝ 反実仮想 「もし～であったならば、～であろうに（実際はそうではない）」

用例

意味 用例 解説
❶反実仮想 「世の中にたえて桜のなかりせば春の心はのどけからまし」 もし桜がなかったら、春の心はのどかだろうに（実際は桜があるから落ち着かない）
❶反実仮想 「鳴かずば撃たれましや」 鳴かなければ撃たれただろうか（鳴いたから撃たれなかった）
❷ためらい 「いかにせまし」 どうしようかしら

入試での注意点

最重要
  1. 「せば…まし」は反実仮想の最頻出パターン
  2. 「せ」＝「き」の未然形 or サ変「す」の未然形
  3. 反実仮想は「事実と逆のこと」を述べている→事実を読み取る問題が頻出

暗記用まとめ

一言で

まし＝反実仮想/ためらいの意志 「せば…まし」→もし～なら～だろうに（事実は逆）。超頻出。

まじの判別

基本データ

まじ: 形容詞シク活用型。終止形に接続（ラ変は連体形に接続）。 意味: 「べし」の打消版。打消推量・打消意志・不可能・打消当然・禁止・不適当の6つ。

6つの意味（「べし」の裏返し）

番号 意味 訳し方 べしとの対応
❶ 打消推量 ～ないだろう 推量の打消
❷ 打消意志 ～するまい、～しないつもりだ 意志の打消
❸ 不可能 ～できないだろう 可能の打消
❹ 打消当然 ～すべきでない 当然の打消
❺ 禁止 ～してはならない 命令の打消
❻ 不適当 ～しないほうがよい 適当の打消

判別法

判別のコツ

「べし」と同じフローチャートを使い、打消に変えるだけ。 一人称→打消意志、二人称→禁止/打消当然、三人称→打消推量。

用例

意味 用例 解説
❶打消推量 「雨は降るまじ」 雨は降らないだろう
❷打消意志 「二度と行くまじ」 もう行くまい
❸不可能 「思ひわくまじきことなり」 区別することができないことだ
❹打消当然 「公に仕ふまじき身なり」 朝廷に仕えるべきでない身だ
❺禁止 「ここに入るまじ」 ここに入ってはならない
❻不適当 「かくては住むまじき所なり」 こうしては住まないほうがよい場所だ

入試での注意点

最重要
  1. 「まじ」＝「べし」の打消で覚える
  2. 判別法は「べし」と同じ（人称＋文脈で絞る）
  3. 「まじき」は連体形として頻出

暗記用まとめ

一言で

まじ ＝ べしの打消版。6つの意味も「べし」の裏返し。 判別は「べし」と同じ方法で、訳を打消に変えるだけ。

む（ん）の判別

基本データ

む（ん）: 四段型活用（○／○／む／む／め／○）。未然形に接続。 意味: 推量・意志・勧誘・婉曲・仮定の5つ。

5つの意味

番号 意味 訳し方 典型場面
❶ 推量 ～だろう 三人称主語
❷ 意志 ～しよう、～つもりだ 一人称主語
❸ 勧誘 ～しませんか 二人称主語
❹ 婉曲 ～ような（こと・もの） 連体形で体言に接続
❺ 仮定 ～としたら 「む＋は」「む＋に」の形

判別フローチャート

Q1: 「む」は連体形で使われているか？（体言を修飾 or 「む＋は/に/こそ」）
  → 「む」＋体言（「む人」「むこと」）→ ❹婉曲「～ような」
  → 「む」＋は/に → ❺仮定「～としたら」
  → No → Q2へ

Q2: 主語は誰か？
  → 一人称（私）→ ❷意志「～しよう」
  → 二人称（あなた）→ ❸勧誘「～しませんか」
  → 三人称（あの人・もの等）→ ❶推量「～だろう」
最重要ルール

主語の人称で判別する。一＝意志、二＝勧誘、三＝推量。 連体形なら婉曲か仮定。

判別の詳細

意味 手がかり 例文
❶推量 三人称主語。「～だろう」で自然。 「雨降らむ」（雨が降るだろう）
❷意志 一人称主語。「～しよう」で自然。 「我行かむ」（私は行こう）
❸勧誘 二人称主語。疑問・呼びかけ。 「いざ、行かむ」（さあ、行きましょう）
❹婉曲 連体形＋体言。「～ような」で自然。 「言はむこと」（言うようなこと）
❺仮定 「む＋は」「む＋に」。「～としたら」。 「行かむには」（行くとしたら）

用例

意味 用例 解説
❶推量 「今日は雨降らむ」 「雨」が主語（三人称）→推量
❷意志 「いづ方へか行かむ」 「私は」が暗黙の主語→意志「どこへ行こうか」
❸勧誘 「いざ、ともに行かむ」 相手への呼びかけ→勧誘
❹婉曲 「聞かむ人、いかに思はむ」 「聞くような人」→婉曲
❺仮定 「世の中に絶えて桜のなかりせば春の心はのどけからまし」→ 「まし」だが、仮定の「む」の参考 仮定

紛らわしいパターン

パターン 判別法
「む」＋「とす」 意志「～しようとする」
「むや」「むか」 疑問＋推量または意志
「なむ」 →「な（強意）＋む（推量）」か「終助詞なむ（願望）」か「係助詞なむ」→ 別項目参照

入試での注意点

最重要
  1. 人称判別が命：一人称＝意志、二人称＝勧誘、三人称＝推量
  2. 連体形の「む」は婉曲か仮定
  3. 「む」の終止形と連体形は同形なので、後続語で判断

暗記用まとめ

一言で

む ＝ 推量/意志/勧誘/婉曲/仮定 一＝意志、二＝勧誘、三＝推量。連体形＝婉曲/仮定。

めり・らしの判別

基本データ

めり: ラ変型活用。終止形に接続（ラ変は連体形）。「目＋あり」が語源。 らし: 無変化（らし／らし／らし／らし／らし／○）。終止形に接続。 共通: どちらも「推定」を表す。

違い

めり らし
意味 見た目の推定（目で見て判断） 根拠のある推定（確かな根拠あり）
確信度 やや低い（～ように見える） 高い（～にちがいない）
訳し方 ～ように見える、～ようだ ～らしい、～にちがいない
語源 「目（め）＋あり」 「ら」＋「し」（形容詞語尾）

判別

助動詞 手がかり 例
めり 視覚的な判断。「見ると～のようだ」。 「人来たるめり」（人が来たように見える）
らし 客観的根拠あり。和歌で多用。 「春らし」→ 春であるらしい（花が咲いている等の根拠あり）

用例

助動詞 用例 解説
めり 「あはれなるめり」 しみじみと趣深いように見える
めり 「人の来たるめり」 人が来たようだ（見た感じ）
らし 「春過ぎて夏来たるらし白妙の衣干すてふ天の香具山」 夏が来たらしい（白い衣が干してある→視覚的根拠）

暗記用まとめ

一言で

めり＝見た目の推定「～ように見える」、らし＝根拠ある推定「～にちがいない」 めりは主観的（目）、らしは客観的（根拠）。

らむの判別

基本データ

らむ: 四段型活用（○／○／らむ／らむ／らめ／○）。終止形に接続（ラ変は連体形）。 意味: 現在推量・原因推量・婉曲の3つ。 ポイント: 「む」の現在版。「む」は未来の推量、「らむ」は現在の推量。

3つの意味

番号 意味 訳し方 典型場面
❶ 現在推量 今ごろ～しているだろう 目に見えない現在のことを推量
❷ 原因推量 どうして～だろう（か） 疑問詞「など」「いかに」と共起
❸ 婉曲 ～ような 連体形で体言を修飾

判別フローチャート

Q1: 疑問詞（「など」「いかに」「いかで」）と共起しているか？
  → Yes → ❷原因推量「どうして～だろうか」
  → No → Q2へ

Q2: 連体形で体言を修飾しているか？
  → Yes → ❸婉曲「～ような」
  → No → ❶現在推量「今ごろ～しているだろう」

用例

意味 用例 解説
❶現在推量 「故郷にて花見らむ」 「故郷で今ごろ花を見ているだろう」
❷原因推量 「などかく思ひ嘆くらむ」 「どうしてこう思い嘆くのだろうか」
❸婉曲 「思ふらむ人」 「思っているような人」

暗記用まとめ

一言で

らむ＝現在推量/原因推量/婉曲。「む」の現在版。 疑問詞あり→原因推量。連体形→婉曲。それ以外→現在推量。

る・らるの判別

基本データ

る: 下二段型活用（れ／れ／る／るる／るれ／れよ）。四段・ナ変・ラ変の未然形に接続。 らる: 下二段型活用（られ／られ／らる／らるる／らるれ／られよ）。上記以外の未然形に接続。 意味: 受身・自発・可能・尊敬の4つ。

4つの意味

番号 意味 訳し方 例
❶ 受身 ～される 人に笑はる
❷ 自発 自然と～される 故郷のことが思ひ出でらる
❸ 可能 ～できる え寝られず
❹ 尊敬 ～なさる、お～になる 帝、仰せらる

判別フローチャート

Q1: 主語は高貴な人物（帝・中宮・大臣など）か？
  → Yes → ❹尊敬（最優先で検討）
  → No → Q2へ

Q2: 「～に…る/らる」の形で、動作の受け手が明確か？
  → Yes → ❶受身
  → No → Q3へ

Q3: 心理・感情の動詞（思ふ・偲ぶ・泣く・眺む等）に付いているか？
  → Yes → ❷自発（「自然と～してしまう」）
  → No → Q4へ

Q4: 打消・反語・疑問の文脈か？（「～ず」「～じ」「え～ず」「いかで」等）
  → Yes → ❸可能（打消を伴う可能が圧倒的）
  → No → 文脈で❶❷❹を再検討

判別の詳細基準

意味 手がかり 共起表現
❶受身 「～に」で動作主が示される。被害の文脈。 「人に」「世に」「風に」
❷自発 心理動詞＋る/らる。「自然と～してしまう」が自然。 「思ひ出でらる」「しのばる」「泣かる」「ながめらる」
❸可能 打消表現と共起。肯定文での可能は稀。 「え～れず」「～れず」「いかで～れむ」
❹尊敬 主語が高貴。他の敬語と併用されることも。 「帝」「御」「給ふ」と共起
最重要ポイント

可能の「る・らる」は打消文でほぼ100%出現する。 肯定文で「る・らる」が出たら、可能ではなく受身・自発・尊敬を考える。 自発は心理動詞限定。「走らる」のような物理動作では自発にならない。

用例

意味 用例 解説
❶受身 「人に笑はる」 「人に」が動作主→受身
❶受身 「女に恥ぢられて」 女性に恥ずかしがられて→受身
❷自発 「秋は来ぬと目にはさやかに見えねども風の音にぞ驚かれぬる」 「驚く」は感覚動詞→自発「自然とはっとさせられた」
❷自発 「いにしへのこと思ひ出でらる」 「思ひ出づ」は心理動詞→自発「自然と思い出される」
❸可能 「え寝られず」 「え～ず」（不可能）の文脈→可能「眠ることができない」
❸可能 「涙のこぼるるを、人に見えじとて」→「見られじ」と同型 打消推量「じ」→可能
❹尊敬 「帝、歌を詠ませ給ふ」→ 「る」が使役か尊敬かは「せ給ふ」で判断 高貴な主語→尊敬
❹尊敬 「大臣、仰せらる」 大臣が主語→尊敬「おっしゃる」

紛らわしいパターン

パターン 判別法
「せ給ふ」「れ給ふ」 「せ/れ」＋「給ふ」→ 尊敬の二重敬語。使役ではない。
「泣かる」 心理動詞→自発「自然と泣けてしまう」。受身ではない。
「見らる」 文脈次第。打消文なら可能、高貴な主語なら尊敬、心理的なら自発。

入試での注意点

最重要
  1. 尊敬は主語で判断（高貴→尊敬）
  2. 自発は心理動詞で判断（思ふ・偲ぶ・泣く・眺む等）
  3. 可能は打消で判断（え～ず、～ず、～じ）
  4. 受身は「～に」で判断（動作主の「に」がある）
  5. 迷ったら消去法。尊敬→自発→可能→受身の順で検討。

暗記用まとめ

一言で

る・らる = 受身/自発/可能/尊敬 判別の優先順位: 尊敬（主語）→ 自発（心理動詞）→ 可能（打消文）→ 受身（～に） 可能は打消文限定。自発は心理動詞限定。

助動詞の接続まとめ

概要

助動詞の判別で最も重要なのは接続。何形に付くかで助動詞を特定できる。

未然形に接続する助動詞

助動詞 意味 語呂合わせ
る・らる 受身・自発・可能・尊敬
す・さす・しむ 使役・尊敬
む（ん）・むず 推量・意志・勧誘・婉曲・仮定
まし 反実仮想・ためらいの意志
じ 打消推量・打消意志
ず 打消
まほし 願望（～たい）

語呂: 「る・らる・す・さす・しむ・ず・じ・む・むず・まし・まほし」→ 「さ（未）らば済ませじ」

連用形に接続する助動詞

助動詞 意味
き 直接過去
けり 間接過去・詠嘆
つ 完了・強意
ぬ 完了・強意
たり 完了・存続
たし 願望（～たい）
けむ 過去推量

語呂: 「き・けり・つ・ぬ・たり・たし・けむ」→ 「用タヌキ（けり）」

終止形に接続する助動詞

助動詞 意味 注意
べし 推量・意志・可能・当然・命令・適当 ラ変は連体形接続
まじ べしの打消 ラ変は連体形接続
らむ 現在推量・原因推量・婉曲 ラ変は連体形接続
めり 推定（見た目） ラ変は連体形接続
らし 推定（根拠あり）
なり（伝聞推定） ～らしい ラ変は連体形接続

語呂: 「べし・まじ・らむ・めり・らし・なり」→ 「止めるべし」

体言・連体形に接続する助動詞

助動詞 意味
なり（断定） ～だ、～である
たり（断定） ～だ、～である
ごとし ～のようだ

特殊な接続

助動詞 接続 語呂
り サ変の未然形・四段の已然形 「さみしい」

入試での注意点

最重要
  1. 接続を覚えれば判別の8割は解ける
  2. 未然形接続 vs 連用形接続の区別が最頻出
  3. 「ぬ」の判別（連用形→完了、未然形→打消）
  4. 「なり」の判別（体言/連体形→断定、終止形→伝聞推定）

暗記用まとめ

一言で

接続が助動詞判別の命。 未然形→る・らる・す・さす・む・まし・じ・ず 連用形→き・けり・つ・ぬ・たり・けむ 終止形→べし・まじ・らむ・めり・らし・なり（伝聞） 体言/連体形→なり（断定）・たり（断定）・ごとし
