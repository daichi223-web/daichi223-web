あく

基本データ
  • 飽く: カ行四段（あか／あき／あく／あく／あけ／あけ）
  • 明く・開く: カ行下二段（あけ／あけ／あく／あくる／あくれ／あけよ） 語源: 「飽く」＝心が一杯になって溢れる。プラス（満足）にもマイナス（嫌気）にもなる

辞書的意味

飽く〔カ行四段〕

番号 意味 典型的な例文 出典
❶ 飽きる、嫌になる 世の中に飽きたる僧 徒然草
❷ 十分に～する、満足する 飽かぬ別れ（＝名残惜しい） 古今和歌集

明く・開く〔カ行下二段〕

番号 意味 典型的な例文 出典
❸ 夜が明ける、年が明ける 夜の明くるも知らず 枕草子
❹ 開く、空く 門をあけさせて 伊勢物語

派生関係：

飽く（原義：心が一杯になる）
  ├→ ❶ 嫌になる（マイナス方向に溢れる）
  └→ ❷ 満足する（プラス方向に満たされる）
❶と❷は正反対の訳になる

「飽かず」＝「満足しない」＝「もっと欲しい・名残惜しい」。マイナスの否定ではなくプラスの否定。

判別基準

意味 文脈上の手がかり 共起しやすい語
❶飽きる 嫌気・倦怠の文脈 「世」「人」「もの」
❷満足する 名残惜しさ・賞賛の文脈 「飽かず」「飽かぬ」が圧倒的に多い
❸夜が明ける 時間経過・朝の場面 「夜」「暁」「鶏」
❹開く 物理的な動作 「門」「戸」「蓋」

判別フローチャート

Q1: 活用形で区別できるか？
  → 四段（あか・あき）→ 「飽く」（❶か❷）
  → 下二段（あけ・あくる）→ 「明く・開く」（❸か❹）

Q2:【飽く】否定形「飽かず」「飽かぬ」か？
  → Yes: ほぼ❷「満足しない＝名残惜しい」
  → No: 文脈で判断（嫌気→❶、賞賛→❷）

Q3:【明く・開く】時間の文脈か？
  → Yes: ❸「夜が明ける」
  → No: ❹「開く」

類義語・対義語

類義語 共通点 相違点
うむ（倦む） 「飽きる」で❶と共通 「倦む」は疲労・倦怠が中心
たる（足る） 「満足する」で❷と共通 「足る」は量的な充足。「飽く」は心理的
対義語 意味 対比のポイント
暮る 日が暮れる ❸「明く」の反対

入試での注意点

最重要

「飽かず」「飽かぬ」は❷「満足しない＝名残惜しい」が圧倒的多数。 「飽きない」と訳すと不自然になる場面が多い。

  • 四段と下二段の活用の区別が出題される
  • 「飽かぬ別れ」「飽かず眺む」のパターンは頻出

関連する慣用表現・連語

表現 意味 入試重要度
飽かず眺む 満足せず（ずっと）眺める ★★★
飽かぬ別れ 名残惜しい別れ ★★★
飽かぬ色 見飽きない美しさ ★★☆
世に飽く 世の中に嫌気がさす → 出家の前兆 ★★☆

暗記用まとめ

一言で

「飽く」＝心が一杯になる → ❶嫌になる ❷満足する。「飽かず」は「名残惜しい」。

活用で区別: 四段（あか・あき）＝飽く、下二段（あけ・あくる）＝明く・開く

あふ

基本データ

品詞: 動詞・ハ行四段 活用: あは／あひ／あふ／あふ／あへ／あへ 語源: 二つのものが一つになる。「会」「合」「逢」「遭」

辞書的意味

番号 意味 典型的な例文 出典
❶ 会う、対面する 人にあふ松の 後撰和歌集
❷ （男女が）結ばれる、契りを結ぶ 夢にだにあふことかたき 古今和歌集
❸ 一致する、調和する、似合う 色のあひたる 枕草子
❹ 遭遇する、出くわす（悪い事態に） 盗人にあひぬ 今昔物語集

派生関係：

原義：二つのものが一つになる
  ├→ ❶ 人と人が会う
  │    └→ ❷ 男女が結ばれる（恋歌での特殊化）
  ├→ ❸ 物と物が合う（一致・調和）
  └→ ❹ 事態に遭う（不本意な遭遇）

判別基準

意味 文脈上の手がかり 共起しやすい語
❶会う 一般的な対面 「人」「友」「見る」
❷結ばれる 恋愛の文脈 「夢」「夜」「契り」「忍ぶ」
❸合う 物の調和・一致 「色」「心」「折」
❹遭う 災難・不本意な事態 「盗人」「難」「憂き目」

判別フローチャート

Q1: 恋愛の文脈か？
  → Yes: ❷「（男女が）結ばれる」
  → No: Q2へ

Q2: 災難・不本意な事態の文脈か？
  → Yes: ❹「遭遇する」
  → No: Q3へ

Q3: 物の一致・調和の文脈か？
  → Yes: ❸「合う」
  → No: ❶「会う」

類義語・対義語

類義語 共通点 相違点
みる 「対面する」で❶と近い 「みる」は視覚。「あふ」は双方向の出会い
ちぎる 「男女の関係」で❷と近い 「ちぎる」は約束・契約。「あふ」は出会いの瞬間
対義語 意味 対比のポイント
わかる（別る） 別れる ❶「会う」の直接的反対

入試での注意点

最重要

恋歌での「あふ」は❷「結ばれる」が基本。「逢ふ瀬」「夢にあふ」は肉体関係を含意。 「会う」と訳すと文脈が薄くなり減点対象になることがある。

  • 掛詞：「あふ坂」（逢坂＝逢ふ）は超頻出
  • 「あふことの」は恋歌の常套句

関連する慣用表現・連語

表現 意味 入試重要度
あふことの絶えて 逢うことが絶えて（百人一首） ★★★
あふ坂の関 掛詞（逢坂＋逢ふ） ★★★
夢にあふ 夢で（恋人に）会う ★★☆
憂き目にあふ つらい目に遭う ★★☆

暗記用まとめ

一言で

「あふ」＝二つが一つになる → 会う・結ばれる・合う・遭う

最重要: 恋歌の「あふ」は「結ばれる」。「あふ坂」は掛詞の定番。

あり

基本データ

品詞: 動詞・ラ行変格活用 活用: あら／あり／あり／ある／あれ／あれ 語源: 存在の根本動詞。古文で最も多義的な動詞の一つ。

辞書的意味

番号 意味 典型的な例文 出典
❶ いる、存在する（人） 人のあるに 枕草子
❷ ある、存在する（物・事） 物のあるかぎり 徒然草
❸ 暮らす、過ごす、生活する 世にあり 源氏物語
❹ （ある状態で）ある（補助動詞的） さてあるべきにあらず 源氏物語

派生関係：

原義：存在する
  ├→ ❶ 人がいる
  ├→ ❷ 物がある
  ├→ ❸ 存在し続ける → 暮らす、過ごす
  └→ ❹ 状態としてある（補助動詞的）

判別基準

意味 文脈上の手がかり 共起しやすい語
❶いる 人物の存在 「人」「女」「僧」
❷ある 物・事の存在 「物」「事」「所」
❸暮らす 時間の経過・生活 「世に」「年ごろ」「宮に」
❹状態にある 形容詞・助動詞の後 「さて」「かく」「〜てあり」

判別フローチャート

Q1: 「世にあり」「宮にあり」など場所＋あり の形か？
  → Yes: ❸「暮らす・生活する」
  → No: Q2へ

Q2: 「〜てあり」「さてあり」の補助動詞的用法か？
  → Yes: ❹「（ある状態で）ある」
  → No: Q3へ

Q3: 主語が人か？
  → Yes: ❶「いる」
  → No: ❷「ある」

類義語・対義語

類義語 共通点 相違点
おはす 「いる」の尊敬語 身分の高い人に使う
はべり 「いる」の丁寧語 会話文・地の文の丁寧表現
さぶらふ 「いる」の謙譲語・丁寧語 お側にお仕えする意も
対義語 意味
なし ない、存在しない

入試での注意点

最重要

「世にあり」は連語として超重要。「俗世にいる＝出家していない」の意。 単に「生きている」と訳すと不十分な場合が多い。

  • 「ありし日」＝「かつての日々」（過去回想の常套句）
  • 「あらまほし」＝「あってほしい＝理想的だ」
  • 補助動詞「〜てあり」は現代語「〜ている」の源

関連する慣用表現・連語

表現 意味 入試重要度
世にあり 俗世で暮らす、出家していない ★★★
ありし日 かつての日々 ★★★
あらまほし あってほしい、理想的だ ★★★
さてあり そのままでいる ★★☆
ありがたし 存在しにくい → めったにない → 素晴らしい ★★★

暗記用まとめ

一言で

「あり」＝存在の万能動詞 → いる・ある・暮らす・状態にある

最重要: 「世にあり」＝俗世にいる（出家していない）。「あらまほし」＝理想的だ。

いづ

基本データ

品詞: 動詞・ダ行下二段 活用: いで／いで／いづ／いづる／いづれ／いでよ 語源: 内側から外側へ出る。物理的な移動から抽象的な「現れ」まで幅広い。

辞書的意味

番号 意味 典型的な例文 出典
❶ 出る、外に出る 門より出でて 竹取物語
❷ 現れる、姿を見せる 月の出でたるに 枕草子
❸ 出発する、出かける 旅に出づ 土佐日記
❹ 生まれる、生じる この世に出でて 源氏物語
❺ （涙が）こぼれる、（声が）出る 涙の出でくるに 更級日記

派生関係：

原義：内から外へ出る
  ├→ ❶ 物理的に出る
  ├→ ❷ 見えなかったものが現れる
  ├→ ❸ 場所から出発する
  ├→ ❹ この世に出現する → 生まれる
  └→ ❺ 体内から出る（涙・声）

判別基準

意味 文脈上の手がかり 共起しやすい語
❶出る 物理的な移動 「門」「家」「外」
❷現れる 自然現象・人物の登場 「月」「日」「人」
❸出発する 旅・移動の文脈 「旅」「都」「道」
❹生まれる 誕生・出自の文脈 「世に」「御子」
❺こぼれる 感情の発露 「涙」「声」「言葉」

類義語・対義語

類義語 共通点 相違点
まかづ 「退出する」で❶に近い 「まかづ」は謙譲。貴人のもとから退出
あらはる 「現れる」で❷と共通 「あらはる」は隠れていたものが露見する
対義語 意味
入る（いる） 入る

入試での注意点

最重要

「出で来（いでく）」＝「やって来る」は超頻出の複合動詞。「出て来る」ではなく「現れる・やって来る」と訳すのが自然。

  • 「世に出づ」＝「世間に知られる」「出世する」の意もある
  • 「言ひ出づ」＝「言い出す、口に出す」

関連する慣用表現・連語

表現 意味 入試重要度
出で来（いでく） やって来る、現れる ★★★
世に出づ 世間に知られる、出世する ★★★
涙の出でくる 涙がこぼれる ★★☆
言ひ出づ 言い出す、口に出す ★★☆

暗記用まとめ

一言で

「いづ」＝内から外へ → 出る・現れる・出発する・生まれる・こぼれる

最重要: 「出で来」＝やって来る。「世に出づ」＝世間に知られる。

いふ

基本データ

品詞: 動詞・ハ行四段 活用: いは／いひ／いふ／いふ／いへ／いへ 語源: 言葉を発する。古文では「名づける」「歌を詠む」など現代語にない用法がある。

辞書的意味

番号 意味 典型的な例文 出典
❶ 言う、話す と人のいひければ 竹取物語
❷ 名づける、～と呼ぶ なよ竹のかぐや姫といふ 竹取物語
❸ 和歌を詠む いひける歌 古今和歌集
❹ 世間で～と言う、評判である 世にいふ 枕草子
❺ （手紙で）書いて伝える 文にいひおこせたる 源氏物語

派生関係：

原義：言葉を発する
  ├→ ❶ 言う、話す
  ├→ ❷ 名前を言う → 名づける
  ├→ ❸ 言葉を作る → 歌を詠む
  ├→ ❹ 人々が言う → 評判である
  └→ ❺ 言葉を送る → 手紙で伝える

判別基準

意味 文脈上の手がかり 共起しやすい語
❶言う 会話・発言の文脈 「と」「けり」「ければ」
❷名づける 命名・呼称の文脈 「〜と」「名」「号」
❸歌を詠む 和歌の文脈 「歌」「返し」
❹評判である 世間の評判 「世に」「人」
❺手紙で伝える 書面・通信の文脈 「文（ふみ）」「おこす」

判別フローチャート

Q1: 「〜といふ」で名前を示しているか？
  → Yes: ❷「名づける」
  → No: Q2へ

Q2: 和歌の前後にあるか？
  → Yes: ❸「歌を詠む」
  → No: Q3へ

Q3: 「世にいふ」「人のいふ」など世間の評判か？
  → Yes: ❹「評判である」
  → No: ❶「言う」

類義語・対義語

類義語 共通点 相違点
のたまふ 「言う」の尊敬語 身分の高い人の発言
まうす 「言う」の謙譲語 目上の人に対して言う
よむ 「歌を詠む」で❸と共通 「よむ」は歌専用

入試での注意点

最重要

「いひける」が和歌の直前にあれば❸「詠んだ」と訳す。「言った」ではない。

  • 「〜といふ所」「〜といふ人」は❷→「〜という所」
  • 「えもいはず」＝「言葉にできないほど（素晴らしい）」は超頻出

関連する慣用表現・連語

表現 意味 入試重要度
いひける歌 詠んだ歌 ★★★
〜といふ 〜という（名前） ★★★
えもいはず 言葉にできないほど ★★★
世にいふ 世間で言われている ★★☆
いひおこす 手紙で言ってよこす ★★☆

暗記用まとめ

一言で

「いふ」＝言葉を使う → 言う・名づける・歌を詠む・評判である・手紙で伝える

最重要: 和歌の前の「いひける」＝「詠んだ」。「えもいはず」＝「言葉にできないほど」。

うつる

基本データ
  • 移る: ラ行四段（うつら／うつり／うつる／うつる／うつれ／うつれ）
  • 映る: ラ行四段（同上。漢字で区別） 語源: 「移」＝場所・状態が変わる。「映」＝光や色が反映する。

辞書的意味

番号 意味 典型的な例文 出典
❶ 移動する、場所を変える 都うつり 方丈記
❷ （色・香が）移る、染まる 花の色のうつりにけりな 古今和歌集
❸ 時が移る、変化する 時うつりて 徒然草
❹ 心が移る、気持ちが変わる 心のうつるに 源氏物語
❺ 映る、映し出される 水にうつる月 古今和歌集

派生関係：

原義：ある場所から別の場所へ
  ├→ ❶ 物理的に移動する
  ├→ ❷ 色・香が別のものに付く
  ├→ ❸ 時間が進む、変化する
  ├→ ❹ 心が他に向く
  └→ ❺ 光が反映する（映る）

判別基準

意味 文脈上の手がかり 共起しやすい語
❶移動する 場所の変更 「都」「所」「家」
❷色・香が移る 衣服・花の文脈 「色」「香（か）」「花」「袖」
❸時が移る 時間経過の文脈 「時」「世」「年」
❹心が移る 恋愛・心情の文脈 「心」「人」
❺映る 光・水面の文脈 「水」「鏡」「月」「影」

類義語・対義語

類義語 共通点 相違点
うつろふ 「色が変わる」で❷❸と共通 「うつろふ」は色褪せる方向に限定されやすい
かはる 「変わる」で❸❹と共通 「かはる」は変化全般。「うつる」は移動のニュアンス

入試での注意点

最重要

百人一首「花の色はうつりにけりな」の「うつる」は❷「色が褪せる」。 「移動した」でも「映った」でもない。

  • 「心うつる」は❹「心変わりする」→ 恋愛文脈で頻出
  • 「うつろふ」との区別も出題される

関連する慣用表現・連語

表現 意味 入試重要度
花の色はうつりにけりな 花の色は褪せてしまった（百人一首） ★★★
心うつる 気持ちが変わる、浮気する ★★★
都うつり 遷都 ★★☆
色うつる 色が移る、染まる ★★☆

暗記用まとめ

一言で

「うつる」＝あるものから別のものへ → 移動・色移り・時の変化・心変わり・映る

最重要: 「花の色はうつりにけりな」＝色が褪せた。「心うつる」＝心変わり。

おこなふ

基本データ

品詞: 動詞・ハ行四段 活用: おこなは／おこなひ／おこなふ／おこなふ／おこなへ／おこなへ 語源: 「行う」。古文では仏道修行の意が最も重要。

辞書的意味

番号 意味 典型的な例文 出典
❶ 仏道修行する、勤行する よくおこなひたる僧 徒然草
❷ 行う、実行する 事をおこなふ 源氏物語
❸ 政治を行う、治める 国をおこなふ 大鏡

派生関係：

原義：物事を実行する
  ├→ ❶ 仏道を実行する → 修行する（古文で最頻出）
  ├→ ❷ 一般的に行う
  └→ ❸ 政治を行う → 治める
古文での最頻出義

現代語では❷「行う」が基本だが、古文では❶「仏道修行する」が圧倒的に多い。 「おこなひ」＝「修行・勤行」（名詞用法）も頻出。

判別基準

意味 文脈上の手がかり 共起しやすい語
❶仏道修行する 僧・仏教の文脈 「僧」「法師」「山」「寺」
❷行う 一般的な行為 「事」「こと」
❸政治を行う 朝廷・政治の文脈 「国」「世」「政（まつりごと）」

  • 映る: ラ行四段（同上。漢字で区別） 語源: 「移」＝場所・状態が変わる。「映」＝光や色が反映する。

辞書的意味

番号 意味 典型的な例文 出典
❶ 移動する、場所を変える 都うつり 方丈記
❷ （色・香が）移る、染まる 花の色のうつりにけりな 古今和歌集
❸ 時が移る、変化する 時うつりて 徒然草
❹ 心が移る、気持ちが変わる 心のうつるに 源氏物語
❺ 映る、映し出される 水にうつる月 古今和歌集

派生関係：

原義：ある場所から別の場所へ
  ├→ ❶ 物理的に移動する
  ├→ ❷ 色・香が別のものに付く
  ├→ ❸ 時間が進む、変化する
  ├→ ❹ 心が他に向く

判別基準

意味 文脈上の手がかり 共起しやすい語

❸時が移る 時間経過の文脈 「時」「世」「年」

入試での注意点

最重要

  • 「心うつる」は❹「心変わりする」→ 恋愛文脈で頻出

おこなふ

基本データ

品詞: 動詞・ハ行四段 活用: おこなは／おこなひ／おこなふ／おこなふ／おこなへ／おこなへ 語源: 「行う」。古文では仏道修行の意が最も重要。

辞書的意味

番号 意味 典型的な例文 出典
❶ 仏道修行する、勤行する よくおこなひたる僧 徒然草
❷ 行う、実行する 事をおこなふ 源氏物語
❸ 政治を行う、治める 国をおこなふ 大鏡

派生関係：

原義：物事を実行する
  ├→ ❶ 仏道を実行する → 修行する（古文で最頻出）
  ├→ ❷ 一般的に行う
  └→ ❸ 政治を行う → 治める

現代語では❷「行う」が基本だが、古文では❶「仏道修行する」が圧倒的に多い。 「おこなひ」＝「修行・勤行」（名詞用法）も頻出。

判別基準

意味 文脈上の手がかり 共起しやすい語

❷行う 一般的な行為 「事」「こと」
❸政治を行う 朝廷・政治の文脈 「国」「世」「政（まつりごと）」

  → No: Q2へ

Q2: 政治・国の文脈か？
  → Yes: ❸「政治を行う」

す（為） 「行う」で❷と共通 「す」はより広い意味の「する」

最重要

古文で「おこなふ」が出たらまず❶「仏道修行する」を疑う。 「行う」と訳して意味が通じない場合は❶。

  • 「おこなひ」（名詞）＝「修行・勤行」が超頻出

暗記用まとめ

一言で

「おこなふ」＝古文では仏道修行するが最頻出。現代語の「行う」とはズレが大きい。

かぬ（兼ぬ）

基本データ

品詞: 動詞・ナ行下二段 活用: かね／かね／かぬ／かぬる／かぬれ／かねよ 語源: 「兼」＝複数を合わせ持つ。転じて「～しきれない」の補助動詞用法が最重要。

辞書的意味

番号 意味 典型的な例文 出典
❶ ～しかねる、～できない（補助動詞） 思ひかねつつ 古今和歌集
❷ 兼ねる、合わせ持つ 二つをかぬ 大鏡
❸ 前もって、あらかじめ（「かねて」の形で副詞的） かねて思ひける 徒然草

派生関係：

原義：複数を合わせ持つ
  ├→ ❶ 合わせ持てない → ～しきれない、～できない
  ├→ ❷ 兼ねる（本動詞）
  └→ ❸ 前もって合わせ持つ → あらかじめ
補助動詞❶が圧倒的に多い

入試で出る「かぬ」は大半が❶の補助動詞用法。 「～かねつ」「～かぬる」＝「～しかねた」「～できない」。

判別基準

意味 文脈上の手がかり 共起しやすい語
❶～しかねる 動詞の連用形＋かぬ 「思ひかぬ」「待ちかぬ」「堪へかぬ」
❷兼ねる 本動詞として 「二つ」「役」
❸あらかじめ 「かねて」の形 「かねて聞く」「かねて思ふ」

判別フローチャート

Q1: 「かねて」の形か？
  → Yes: ❸「あらかじめ」
  → No: Q2へ

Q2: 動詞の連用形＋「かぬ」の形か？
  → Yes: ❶「～しかねる、～できない」
  → No: ❷「兼ねる」

類義語・対義語

類義語 共通点 相違点
えず（得ず） 「できない」で❶と共通 「え～ず」は不可能の副詞＋打消。「かぬ」は動詞の補助
わぶ 「つらくて耐えられない」で❶に近い 「わぶ」は感情的な苦しさ

入試での注意点

最重要

「待ちかねて」「思ひかねて」は❶「～しきれなくて」が正解。 「前もって待って」と❸で訳すと誤り。前後の動詞の有無で判別。

  • 「かねて」単独なら❸「あらかじめ」
  • 「動詞＋かねて」なら❶「～しきれなくて」
  • 「かねてより」＝「以前から」（❸の強調形）

一言で

くだる

基本データ

品詞: 動詞・ラ行四段 活用: くだら／くだり／くだる／くだる／くだれ／くだれ 語源: 上から下へ移動する。都（上）から地方（下）への移動が最重要。

辞書的意味

番号 意味 典型的な例文 出典
❶ 都から地方へ行く（下向する） 東（あづま）にくだる 伊勢物語
❷ 下がる、降りる（物理的） 山をくだる 徒然草

❹ （命令・宣旨が）下される 宣旨くだる 源氏物語

派生関係：

原義：上から下へ
  ├→ ❶ 都（上位）→ 地方（下位）＝下向
  ├→ ❷ 物理的に下がる
  ├→ ❸ 身分が下がる
  └→ ❹ 上位から命令が下る

古文では「都＝上」「地方＝下」。都から出ることは「くだる」、都に行くことは「のぼる」。

意味 文脈上の手がかり 共起しやすい語
❶都から地方へ 地名・方角の移動 「東」「西国」「任地」
❷物理的に降りる 地形・建物 「山」「坂」「階（はし）」

❹命令が下る 朝廷・権力者 「宣旨」「仰せ」「勅」

類義語 共通点 相違点
おる（降る） 「降りる」で❷と共通 「おる」は乗り物から降りるなど
まかる 「都から退出する」で❶に近い 「まかる」は謙譲語

のぼる 都に上る、上京する

入試での注意点

最重要

「くだる」は❶「都から地方へ行く」が最頻出。「下りる」と物理的に訳すと文脈に合わない場合は❶。

  • 「のぼる」との対比で出題されることが多い

一言で

「くだる」＝上から下へ → 都→地方が最重要。「のぼる」の反対。

最重要: 「東くだり」＝都から東国へ。「都＝上、地方＝下」の古文常識。

❺番をする 警備・管理 「門」「関」「番」
和歌での掛詞

「もる」は「漏る」と「守る」を掛けることがある。 「木の間よりもる月」＝木の間から漏れる月＋（恋人を）見守る月。

類義語・対義語

類義語 共通点 相違点
まもる（守る） 「見守る」で❹と共通 「まもる」は「目＋守る」＝じっと見つめる

入試での注意点

最重要

「もる」は「漏る」と「守る」の区別が第一。 和歌では掛詞の可能性を常に考慮する。

  • 「雨のもる」＝雨が漏る（❶）→ 恋歌では涙の比喩にも
  • 「木の間よりもる月影」＝月光が漏れる（❷）
  • 「まもる」との混同に注意（「まもる」は「見つめる」が中心義）

暗記用まとめ

一言で

「もる」＝漏る（漏れる・光が差す・秘密が漏れる）＋守る（見守る・番をする）。

最重要: 和歌で「漏る」＋「守る」の掛詞に注意。「まもる」は別語。

よる

基本データ
  • 寄る: ラ行四段（よら／より／よる／よる／よれ／よれ）
  • 因る・依る・拠る: ラ行四段（同上） 語源: 「寄」＝ある方向に近づく。「因・依」＝何かに基づく。

辞書的意味

寄る〔ラ行四段〕

番号 意味 典型的な例文 出典
❶ 寄る、近づく そばによる 源氏物語
❷ 立ち寄る、訪問する 宿による 土佐日記
❸ 寄り添う、親しくなる よりそふ 源氏物語

因る・依る〔ラ行四段〕

番号 意味 典型的な例文 出典
❹ ～による、～のために（原因） 風により 徒然草
❺ ～に基づく、よりどころとする 先例による 大鏡

派生関係：

「寄る」系：近づく
  ├→ ❶ 物理的に近づく
  ├→ ❷ 訪問する
  └→ ❸ 心理的に近づく → 寄り添う
「因る」系：基づく
  ├→ ❹ 原因・理由
  └→ ❺ 根拠・典拠

判別基準

意味 文脈上の手がかり 共起しやすい語
❶近づく 移動・接近 「そばに」「かたへに」
❷立ち寄る 旅・訪問 「宿」「所」
❸寄り添う 人間関係 「添ふ」「かたみに」
❹原因 理由の提示 「～に」「ゆゑ」
❺根拠 典拠・根拠 「先例」「古き」
格助詞「より」との区別

格助詞「より」（＝～から、～よりも）は動詞「よる」の連用形ではない。 「京より来る」の「より」は格助詞。

類義語・対義語

類義語 共通点 相違点
ちかづく（近づく） 「近づく」で❶と共通 「ちかづく」は距離の接近、「よる」はそばに身を寄せる
たづぬ（訪ぬ） 「訪問する」で❷と共通 「たづぬ」は人を目的に訪ねる
対義語 意味
はなる（離る） 離れる。❶❸の反対

入試での注意点

最重要

「よる」の❹「原因」用法は「～によりて」の形で超頻出。 格助詞「より」と動詞「よる」の区別も出題される。

  • 「ゐよりて」＝座って近寄って（❶）
  • 「風によりて」＝風のために（❹）
  • 「若くより」の「より」は格助詞（＝若い時から）であり動詞ではない

わたる（渡る）

基本データ

品詞: 動詞・ラ行四段 活用: わたら／わたり／わたる／わたる／わたれ／わたれ 語源: 「渡」＝水を渡って向こう側へ行く。物理的な横断→空間的な移動→時間の経過→一面に広がる。

辞書的意味

番号 意味 典型的な例文 出典
❶ 渡る、横切る 川をわたる 伊勢物語
❷ 行く、来る、いらっしゃる 殿にわたる 源氏物語
❸ 暮らす、生活する 世をわたる 方丈記
❹ 一面に広がる、及ぶ 霧わたる 古今和歌集
❺ 〔補助動詞〕ずっと～し続ける 思ひわたる 源氏物語

派生関係：

原義：水を渡って向こうへ
  ├→ ❶ 物理的に横断する
  ├→ ❷ 空間を移動する → 行く・来る（尊敬語的用法あり）
  ├→ ❸ 時間を渡る → 暮らす・過ごす
  ├→ ❹ 空間全体に渡る → 広がる
  └→ ❺ 時間を通じて渡る → ずっと～する（補助動詞）

判別基準

意味 文脈上の手がかり 共起しやすい語
❶渡る 水・地形の横断 「川」「橋」「海」
❷行く・来る 貴人の移動（尊敬） 「殿に」「あちらに」
❸暮らす 生活・人生 「世」「年ごろ」
❹広がる 自然描写 「霧」「雲」「雪」「一面に」
❺補助動詞 「連用形＋わたる」 「思ひわたる」「嘆きわたる」
補助動詞の見分け方

「動詞の連用形＋わたる」の形なら❺。 「思ひわたる」＝ずっと思い続ける。「降りわたる」＝一面に降り続ける。

類義語・対義語

類義語 共通点 相違点
すむ（住む） 「暮らす」で❸と共通 「住む」は定住、「わたる」は時の流れの中で暮らす
おはす 「いらっしゃる」で❷と共通 「おはす」は存在の尊敬語、「わたる」は移動

入試での注意点

最重要

補助動詞❺「～しわたる」が最頻出。「思ひわたる」＝「ずっと思い続ける」。 ❷の尊敬語的用法（貴人の移動に使う）も注意。

  • 「霧わたる」＝霧が一面に広がる（❹）→ 自然描写の定番
  • 「世をわたる」＝世の中を生きていく（❸）
  • 「殿にわたりたまふ」＝殿に移動なさる（❷＋尊敬）
  • 「嘆きわたる」＝ずっと嘆き続ける（❺）

関連する慣用表現・連語

表現 意味 入試重要度
思ひわたる ずっと思い続ける ★★★
霧わたる 霧が一面に広がる ★★★
世をわたる 世の中を生きていく ★★☆
わたりたまふ いらっしゃる（尊敬） ★★☆

暗記用まとめ

一言で

「わたる」＝渡る→行く→暮らす→広がる＋補助動詞「ずっと～する」。

最重要: 「連用形＋わたる」＝「ずっと～し続ける」。「霧わたる」＝一面に広がる。
