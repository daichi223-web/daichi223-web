みる

基本データ

品詞: 動詞・マ行上一段 活用: み／み／みる／みる／みれ／みよ 漢字: 「見る」「観る」 語源: 目で対象を捉える → 見る → 世話をする → 男女が会う。 ※ 上一段活用の代表語（ひいきにみゐる）。

辞書的意味

番号 意味 典型的な例文 出典
❶ 見る、目にする 月をみる —
❷ 世話をする、面倒をみる 子をみる —
❸ 結婚する（男女が関係する） 女をみむとて 伊勢物語
❹ 経験する 憂き目をみる —
❺ 試みる（補助動詞「〜てみる」） こころみる —

派生関係：

原義：目で対象を捉える → 見る
  ├→ ❶ 目で見る → 見る・眺める
  ├→ ❷ 目をかける → 世話をする・面倒をみる
  ├→ ❸ 互いに見る → 男女が会う → 結婚する
  ├→ ❹ 目に遭う → 経験する（「目をみる」）
  └→ ❺ ためしに見る → 試みる（補助動詞化）
※ ❸「みる」＝結婚する は古語特有の重要義
※ 「こころみる」＝試みる（心＋見る）
※ 「かいまみる（垣間見る）」＝ のぞき見る
現古の違い

現代語「見る」＝ 目で見る（❶のみ） 古語「みる」＝ ❶見る＋❷世話する＋❸結婚する＋❹経験する → ❸「女をみる」＝結婚する は古語特有の超重要義。 → 「見る」＝「会う」＝「結婚する」が古語の発想。 ※ 「こころみる（試みる）」は複合動詞。

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶見る 視覚の場面。 「月」「花」「ながむ」
❷世話する 介護・養育の場面。 「子」「母」「病」
❸結婚する 男女関係の場面。 「女」「妻」「男」「通ふ」
❹経験する 苦難の場面。 「憂き目」「つらき目」
❺試みる 「〜てみる」の補助動詞用法。 「て」「こころ」
判別のコツ

男女の場面 → ❸ 結婚する（最重要） 「〜を見る」で苦難 → ❹ 経験する 介護・養育 → ❷ 世話する → 「かいまみる」＝ のぞき見る（❶の派生）

用例（意味番号付き）

意味 用例 文脈解説
❶見る 「花の色は移りにけりないたづらにわが身世にふるながめせしまに」→花を見る場面 花を見ながら物思いにふける→視覚＋感慨
❸結婚する 「昔、男ありけり。女をみむとて」（伊勢物語） 女と結婚しようとして→男女の出会い
❹経験する 「憂き目をみる」 つらい経験をする→苦難

入試での注意点

最重要

「みる」＝「見る・世話する・結婚する・経験する」。マ行上一段。 ❸「みる」＝結婚する が入試で超重要。 上一段活用の代表語（「ひいきにみゐる」で暗記）。

  • 「女をみる」＝ 結婚する
  • 「かいまみる（垣間見る）」＝ のぞき見る
  • 「こころみる（試みる）」＝ 試す

覚え方のコツ

覚え方

「みる」＝ 見る → 目をかける → 世話する → 男女が会う → 結婚する → 「見る」から「結婚」まで意味が広がる。

イメージ: 見る → 気にかけて世話する → 男女が「見合う」→ 結婚

ポイント: 「女をみる」＝結婚。上一段「ひいきにみゐる」。
