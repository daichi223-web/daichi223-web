おぼし

基本データ

品詞: 形容詞・シク活用 活用: おぼしく／おぼしく／おぼし／おぼしき／おぼしけれ／○ 語源: 「覚し（おぼし）」＝ 覚える・思われる → 〜と思われる、〜らしい。 ※ 尊敬動詞「思す（おぼす）」とは別語。

辞書的意味

番号 意味 典型的な例文 出典
❶ 〜と思われる、〜らしい 忠度の声とおぼしくて 平家物語
❷ 思っている（こと） おぼしきこと言はぬは 大鏡

派生関係：

原義：覚し → 心に覚える・思う
  ├→ ❶ 客観的推量 → 〜と思われる、〜らしい、〜と見える（最重要義）
  └→ ❷ 主観的内面 → 思っている（こと）、心に抱く
注意

「おぼし」≠「おぼつかなし」 「おぼつかなし」は「はっきりしない・不安だ」で全く別語。 「おぼし」は「思われる・らしい」。

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶〜と思われる 推量・判断の場面。「Aと思われるB」の形。連体形「おぼしき」が圧倒的に多い。 「〜と」「ところ」「もの」「声」
❷思っている 内心を述べる場面。「おぼしきこと」＝思っていること。 「こと」「言ふ」
判別のコツ

「Aとおぼしき＋名詞」＝ Aと思われる〜 → ❶ 「おぼしきこと」＝ 思っていること → ❷ → 連体形「おぼしき」で使われることが圧倒的に多い。

用例（意味番号付き）

意味 用例 文脈解説
❶〜と思われる 「例の家とおぼしきところにものしたり」（蜻蛉日記） いつもの家と思われるところに行った→推量的判断
❶〜らしい 「秀歌とおぼしきを百余首書き集められたる巻き物を」（平家物語） 秀歌と思われるものを百余首書き集めた巻物を→選別の判断
❶〜と思われる 「忠度の声とおぼしくて」（平家物語） 忠度の声と思われる声で→人物推定
❶〜らしい 「京より、叔母とおぼしき人ものしたり」（蜻蛉日記） 京から、叔母と思われる人がやって来た→人物推定
❷思っている 「おぼしきこと言はぬは、げにぞ腹ふくるる心地しける」（大鏡） 思っていることを言わないのは、本当に腹が膨れる気持ちがした→内心の抑圧

類義語・対義語

類義語 共通点 相違点
めく 「〜らしい」で❶と近い 「めく」は接尾語、「おぼし」は独立の形容詞
覚ゆ 「思われる」で❶と近い 「覚ゆ」は動詞、「おぼし」は形容詞で連体修飾に使う
対義語 意味
しるし はっきりしている、明らかだ

入試での注意点

最重要

「おぼし」＝「〜と思われる・〜らしい」。 「おぼつかなし（はっきりしない）」と混同しないこと。

  • 平家物語「忠度の都落ち」の用例が頻出
  • 大鏡「おぼしきこと言はぬは」は超有名一節
  • 連体形「おぼしき」＋名詞の形が最も多い

覚え方のコツ

覚え方

「覚し」＝ 覚える → 〜と思われる、〜らしい → 「覚え」のある → それと覚える → 〜らしい。

イメージ: 「忠度の声とおぼしくて」→ 忠度の声らしくて → 推量判断

ポイント: 「おぼしき＋名詞」で「〜と思われる〜」が最頻出パターン。「おぼしきこと言はぬは腹ふくるる」を丸暗記。

暗記用まとめ

一言で

「おぼし」＝〜と思われる・〜らしい。「覚し」で推量的判断を表す。

最重要: 「おぼつかなし」とは別語。「おぼしき＋名詞」＝〜と思われる〜。大鏡の「おぼしきこと言はぬは腹ふくるる」。
