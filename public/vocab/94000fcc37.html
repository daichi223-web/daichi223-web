つ・ぬの判別

基本データ

つ: 下二段型活用（て／て／つ／つる／つれ／てよ）。連用形に接続。 ぬ: ナ変型活用（な／に／ぬ／ぬる／ぬれ／ね）。連用形に接続。 意味: 完了・強意の2つ。

意味

番号 意味 訳し方
❶ 完了 ～た、～てしまった
❷ 強意（確述） きっと～、必ず～

「つ」と「ぬ」の使い分け

つ ぬ
ニュアンス 意志的・人為的 自然的・無意志的
典型動作 「書きつ」「取りつ」「言ひつ」 「散りぬ」「暮れぬ」「死にぬ」
覚え方 人がやること＝つ 自然に起こること＝ぬ

強意の判別

「つ」「ぬ」が強意になるのは、推量の助動詞の直前のみ。

パターン 意味
て＋む → 「てむ」 きっと～だろう、～してしまおう
な＋む → 「なむ」 きっと～だろう
に＋けり → 「にけり」 ～してしまった（完了）
て＋む＋ず → 「てじ」 きっと～ないだろう
ぬ＋べし → 「ぬべし」 きっと～にちがいない、今にも～しそうだ
つ＋べし → 「つべし」 きっと～にちがいない
最重要ルール

「つ/ぬ」＋推量系助動詞（む・べし・らむ・けむ・まし・じ）＝ 強意 それ以外は完了。

判別フローチャート

Q1: 「つ/ぬ」の直後に推量系助動詞（む・べし・らむ・まし・じ等）があるか？
  → Yes → ❷強意「きっと～」
  → No → ❶完了「～た、～してしまった」

用例

意味 用例 解説
❶完了（つ） 「書きつ」 書いた・書いてしまった（意志的完了）
❶完了（ぬ） 「花散りぬ」 花が散った・散ってしまった（自然的完了）
❷強意 「死にてむとす」 きっと死んでしまおうとする（て＋む→強意）
❷強意 「風立ちぬべし」 きっと風が立つにちがいない（ぬ＋べし→強意）
❷強意 「泣きぬべし」 今にも泣いてしまいそうだ

紛らわしい「ぬ」の判別

→ 詳しくは ぬ・ねの判別 を参照。

形 正体 判別法
「散りぬ」 完了「ぬ」終止形 連用形＋ぬ → 完了
「散らぬ」 打消「ず」連体形 未然形＋ぬ → 打消

入試での注意点

最重要
  1. つ＝意志的、ぬ＝自然的（ニュアンスの違い）
  2. 「つ/ぬ」＋推量系＝強意（それ以外は完了）
  3. 「ぬ」と打消「ず」の連体形「ぬ」の区別が超頻出

暗記用まとめ

一言で

つ＝意志的完了、ぬ＝自然的完了。推量の直前なら強意。 「ぬべし」「てむ」「なむ」→ 強意。それ以外→ 完了。
