よし

基本データ

品詞: 形容詞ク活用「よし（良し・善し）」／名詞「よし（由・因・縁）」 活用（形容詞）: よく／よく／よし／よき／よけれ／○ 漢字: 「良し・善し」（形容詞）／「由・因・縁」（名詞） 語源: 形容詞「よし」は上代からの基本形容詞で「好ましい状態」。名詞「よし」は動詞「寄（よ）す」と同根で「寄りどころ」→「事情・縁・方法」と多義に派生。同音異義で品詞判別が重要。

辞書的意味

番号 意味 典型的な例文 出典
❶ 【形容詞】よい、好ましい、すぐれている よき日出で来て 土佐日記
❷ 【形容詞】身分が高い、教養がある、立派な よき人はあやしきことを語らず 徒然草
❸ 【名詞】縁、ゆかり、関係 しるよしして 伊勢物語
❹ 【名詞】方法、手段、手立て 人に知られでくるよし 後撰和歌集
❺ 【名詞】風情、趣、由緒 故びよしある所 平家物語
❻ 【名詞】事情、いきさつ、わけ そのよしを書きつく 土佐日記
❼ 【名詞】旨、趣旨、内容 燃やすべきよし 竹取物語
❽ 【名詞】そぶり、ふり、様子 心得たるよしして 徒然草

派生関係：

【形容詞「よし（良し）」】
原義：好ましい状態
  ├→ ❶ 事物が好ましい → よい
  └→ ❷ 身分・教養が高く立派 → 身分が高く教養のある（中古以降頻出）

【名詞「よし（由）」】
原義：寄りどころ・由来
  ├→ ❸ 寄りどころの縁 → 縁・ゆかり
  ├→ ❹ 手段として寄るもの → 方法・手段
  ├→ ❺ 由来ある美 → 風情・趣
  ├→ ❻ 事の由来 → 事情・わけ
  ├→ ❼ 伝達内容 → 旨・趣旨
  └→ ❽ 表面的な見せかけ → そぶり・ふり
※ 「〜よしあり」「〜よしして」など多様な連体修飾で出る
注意

「よし」は超多義の重要語。まず 品詞（形容詞か名詞か） を判別。 名詞「よし（由）」は類語「ゆゑ（故）」と対比される： ・「ゆゑ」＝ 本質的な理由・由緒・風情（格調） ・「よし」＝ 表面的な事情・方法・そぶり（実用・口実） 文脈により「事情」「旨」「そぶり」などを訳し分ける。

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶よい（形） 天気・物事の好ましさ 「日」「こと」「物」
❷身分高い（形） 人物の身分・教養 「人」「家」「生まれ」
❸縁（名） 土地・人との関係 「しる」「して」「ゆかり」
❹方法（名） 手段・てだて 「〜べき」「もがな」「なし」
❺風情（名） 景色・趣のある物 「あり」「古びて」「木かげ」
❻事情（名） 経緯・内容の説明 「書き付く」「語る」「の」
❼旨（名） 命令・伝達の中身 「仰す」「申す」「べき」
❽そぶり（名） 見せかけの態度 「して」「顔」「気色」
判別のコツ

活用するか否かで形容詞（❶❷）と名詞（❸〜❽）を判別。 名詞内部の判別は連体修飾の関係で： - 「〜するよし」＝ ❼旨・❻事情 - 「〜しるよし」＝ ❸縁・❻事情 - 「〜あるよし／ある所」＝ ❺風情 - 「〜したるよしして」＝ ❽そぶり 類語「ゆゑ」とペアで出たら「よし」は二次的・表層的意味。

用例（意味番号付き）

意味 用例 文脈解説
❶よい 「風も吹かず、よき日出で来て、漕ぎ行く」（土佐日記） 風も吹かず、よい日和になってきて、漕ぎ進む→航海日和
❷身分高い 「よき人はあやしきことを語らず」（徒然草） 身分が高く教養のある人は不思議なことは語らない→教訓
❸縁 「平城の京、春日の里に、しるよしして、狩りに往にけり」（伊勢物語） 奈良の都の春日の里に、土地を領有する縁で、鷹狩りに行った→初段・芥川
❹方法 「人に知られでくるよしもがな」（後撰和歌集） 人に知られないで逢いに来る方法があればなあ→恋の願い
❺風情 「ふりにける岩の絶え間より、落ちくる水の音さへ、故びよしある所なり」（平家物語） 古びた岩のすき間から、落ちてくる水の音までが、由緒ありげで風情のある所である
❻事情 「十二月の二十日あまり一日の日の戌の時に、門出す。そのよし、いささかにものに書きつく」（土佐日記） 12月21日の午後8時ごろに出発する。その事情を少しばかり紙に書きつける→日記の冒頭
❼旨 「火をつけて燃やすべきよし仰せ給ふ」（竹取物語） 火をつけて燃やせという旨をご命令になる→不死の薬を焼く場面
❽そぶり 「心得たるよしして、賢げにうちうなづき、ほほ笑みてゐたれど、つやつや知らぬ人あり」（徒然草） わかったそぶりをして、賢そうにうなずき、ほほえんでいるけれど、まったくわかっていない人もいる→見栄っ張り批判

類義語・対義語

類義語 共通点 相違点
ゆゑ（故） 理由・由緒・風情で重なる 「ゆゑ」は本質的・格調高い、「よし」は表面的・実用的
ことわり（理） ❻と近く「道理」 「ことわり」は論理、「よし」は事情
すべ ❹と近く「方法」 「すべ」はより技術的、「よし」は機会・手段
けはひ ❽と近く「様子」 「けはひ」は気配、「よし」は装ったそぶり
対義語 意味
あし（悪し） 悪い（❶の反対）
いやし（卑し） 身分が低い（❷の反対）
よしなし 取るに足りない・方法がない

入試での注意点

最重要

「よし」は 形容詞か名詞か の品詞判別が最初の関門。 名詞「よし」は ❸縁／❹方法／❺風情／❻事情／❼旨／❽そぶり の6義を文脈で訳し分ける。 類語「ゆゑ」との使い分けが判別問題で頻出。

  • 形容詞❷「よき人」＝ 身分が高く教養のある人（徒然草頻出）
  • 「〜すべきよし仰す」＝ 〜せよという旨をご命令になる（❼）
  • 「しるよしして」＝ 領有する縁で（❸・伊勢物語初段）
  • 「〜たるよしして」＝ 〜したそぶりをして（❽）
  • 「そのよし書きつく」＝ その事情を書きつける（❻）

覚え方のコツ

覚え方

形容詞「よし」＝ 好ましい → よい・身分高い 名詞「よし（由）」＝ 寄りどころ → 縁・方法・風情・事情・旨・そぶり

イメージ: 名詞「よし」は「寄（よ）り処」→ 物事の寄りどころ＝全ての背景要素 → 縁も方法も風情も事情も「よし（寄り処）」

ポイント: 8義もあるが、形容詞2義＋名詞6義で整理。類語「ゆゑ」は本質、「よし」は表層。

暗記用まとめ

一言で

形容詞「よし」＝❶よい／❷身分が高い。 名詞「よし（由）」＝❸縁／❹方法／❺風情／❻事情／❼旨／❽そぶり。

最重要: 品詞判別＋名詞の6義判別。類語「ゆゑ」との対比で覚える。超多義語。
