そうす（奏す）

基本データ

品詞: 動詞・サ行変格 活用: そうせ／そうし／そうす／そうする／そうすれ／そうせよ 漢字: 「奏す」 語源: 漢語「奏（そう）」＋「す（サ変）」→ 天子に申し上げる。 ※ 天皇・上皇に申し上げる専用の謙譲語。相手が天皇に限定される点が最重要。

辞書的意味

番号 意味 典型的な例文 出典
❶ 天皇に申し上げる（謙譲語） 帝にそうす —
❷ 演奏する 楽をそうす —

派生関係：

【奏す系】
  └→ ❶ 天皇のお耳に入れる → 天皇に申し上げる（謙譲語）
【奏す系（音楽）】
  └→ ❷ 音楽を奏でる → 演奏する
※ 「いふ」の謙譲語体系:
  - 天皇に → 「そうす（奏す）」
  - 中宮・東宮に → 「けいす（啓す）」
  - 一般貴人に → 「まうす（申す）」「きこゆ」
※ 漢語サ変: そうせ・そうし・そうす・そうする・そうすれ・そうせよ
注意

❶は相手が天皇（上皇）に限定される謙譲語。 天皇に → そうす（奏す） 中宮・東宮に → けいす（啓す） 一般貴人に → まうす（申す）/きこゆ → 「まうす」と異なり、相手が天皇でなければ「そうす」は使えない。

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶天皇に申し上げる 天皇への奏上場面。 「帝」「上（うへ）」「御前」
❷演奏する 音楽の場面。 「楽」「笛」「琴」
判別のコツ

天皇が相手 → ❶ 申し上げる 楽器・音楽 → ❷ 演奏する → ❶が入試では重要。

用例（意味番号付き）

意味 用例 文脈解説
❶天皇に申し上げる 「大臣帝にそうして」（竹取物語） 大臣が帝に奏上して→天皇への進言
❷演奏する 「楽をそうす」 音楽を演奏する→音楽の場面

類義語 共通点 相違点

入試での注意点

最重要

「そうす（奏す）」＝「天皇に申し上げる（謙譲語）・演奏する」。サ行変格。 相手が天皇に限定される点が最重要。 「そうす」「けいす」「まうす」の使い分けが出題される。

  • 天皇 → そうす（奏す）
  • 中宮・東宮 → けいす（啓す）
  • 一般貴人 → まうす（申す）/きこゆ
  • サ変: そうせ・そうし・そうす・そうする・そうすれ

覚え方のコツ

覚え方

「そうす」＝ 奏す → 天皇に奏上する → 天皇専用の「申す」 → 「そうす＝天皇、けいす＝中宮、まうす＝一般」の三段階。

語呂: 「奏（そう）は天、啓（けい）は宮、申（まう）は人」

ポイント: 相手限定。天皇以外に「そうす」は使えない。
