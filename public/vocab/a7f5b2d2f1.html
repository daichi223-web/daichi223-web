おこたる

基本データ

品詞: 動詞・ラ行四段 活用: おこたら／おこたり／おこたる／おこたる／おこたれ／おこたれ 漢字: 「怠る」 語源: 「怠る」＝ 力が抜ける → 緩む → 病気が緩む → 回復する。

辞書的意味

番号 意味 典型的な例文 出典
❶ （病気が）よくなる、回復する 心地おこたりて 古今集
❷ 怠る、なまける、油断する おこたりなく 徒然草

派生関係：

原義：力が抜ける → 緩む
  ├→ ❶ 病気の勢いが緩む → 回復する・よくなる（古語独特・最重要義）
  └→ ❷ 気が緩む → 怠る・なまける・油断する（現代語と同じ）
現古の違い

現代語「怠る」＝ なまける（❷のみ） 古語「おこたる」＝ ❶（病気が）よくなる が第一義 → 「心地おこたりて」を「気分を怠って」と訳すと完全な誤り。「気分がよくなって」が正しい。 ※ 古文で「おこたる」の直前に「病」「心地」「わづらふ」があれば❶確定。

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶病気がよくなる 病気・体調の話題。病を患った後の回復場面。 「心地」「病」「わづらふ」「なやむ」
❷怠る 仕事・修行・義務の場面。気を抜く場面。 「行ひ」「つとめ」「心」
判別のコツ

病気・体調の文脈 → ❶ 回復する 義務・修行の文脈 → ❷ 怠る → 古文で「おこたる」が出たら、直前に病気関連語がないか必ず確認。あれば❶。

用例（意味番号付き）

意味 用例 文脈解説
❶回復する 「心地おこたりてのち、とぶらへりければ」（古今集） 気分がよくなった後で、見舞ってくれたので→病からの回復
❶回復する 「おなじ少将、病にいといたうわづらひて、少しおこたりて」（大和物語） 同じ少将が、病にひどく患って、少し回復して→重病からの快方
❶回復する 「日ごろ月ごろしるきことありてなやみわたるが、おこたりぬるもうれし」（枕草子） 何日も何か月もはっきりした症状で悩み続けていたのが、回復したのもうれしい
❶回復する 「読経修法などして、いささかおこたりたるやうなれば」（蜻蛉日記） 読経や加持祈祷などして、少し回復したようなので
❷怠る 「あだあだしく心弱くなりおきにけるわがおこたりに」（源氏物語） 浮気っぽく心弱くなってしまった自分の油断に→光源氏の反省

類義語・対義語

類義語 共通点 相違点
いゆ（癒ゆ） 「回復する」で❶と同義 「いゆ」は完全に治る、「おこたる」は症状が緩む
なほる 「よくなる」で❶と近い 「なほる」は元通りになる、「おこたる」は病状の緩和
対義語 意味
わづらふ 病に苦しむ、患う
なやむ 病に悩む

入試での注意点

最重要

「おこたる」＝「（病気が）よくなる・回復する」が古語の第一義。 「怠る・なまける」だけで訳すと致命的な誤訳。 病気の文脈では必ず❶「回復する」。

  • 枕草子・蜻蛉日記・大和物語で頻出
  • 「心地おこたる」＝気分がよくなる（最頻出パターン）
  • 現古異義語の超重要語
  • 「いゆ（癒ゆ）」と並ぶ回復の動詞

覚え方のコツ

覚え方

「おこたる」＝ 病気が「怠けて」くれる → 病気がサボる → 回復！ → 病気が怠けて（力を抜いて）くれた → 体が楽になった。

イメージ: 病気の神様が怠けてくれた → 病気がよくなった → おこたる

ポイント: 病気の文脈＝回復。「心地おこたる」＝気分がよくなる。現古異義語の代表。
