あたらし

基本データ

品詞: 形容詞・シク活用 活用: あたらしく／あたらしく／あたらし／あたらしき／あたらしけれ／○ 語源: 「あたら（惜しい）」＋「し」。もったいない、惜しいの意。

辞書的意味

番号 意味 典型的な例文 出典
❶ 惜しい、もったいない あたらしき人 源氏物語
❷ （惜しいほど）立派だ、素晴らしい あたらしき御ありさま 源氏物語

派生関係：

原義：惜しい、もったいない
  ├→ ❶ 才能や美しさが失われて惜しい
  └→ ❷ 惜しいほど立派だ（賞賛のニュアンス）
現代語との決定的な違い

現代語「新しい」＝ 古語「あらたし」（ラ行） 古語「あたらし」＝ 惜しい、もったいない 「あらたし」が音変化で「あたらし」になったのは近世以降。古文で「あたらし」と出たら「新しい」ではなく「惜しい」。

判別基準

意味 文脈上の手がかり 共起しやすい語
❶惜しい 不遇・喪失の場面 「人」「才」「容貌」
❷立派だ 人物評・賞賛 「御ありさま」「さま」

類義語・対義語

類義語 共通点 相違点
くちをし（口惜し） 「惜しい」で❶と共通 「くちをし」は残念・悔しいの感情
をし（惜し） 「惜しい」で❶と共通 「をし」は失いたくないの意

入試での注意点

最重要

「あたらし」＝「惜しい」。「新しい」と訳したら不正解。 古文で「新しい」は「あらたし」。

  • 「あたらしき人」＝惜しいほど立派な人（才能が活かされない）
  • 「あたら口惜しけれ」＝惜しくも残念だ（❶＋くちをし）
  • 副詞「あたら」（＝惜しいことに）も同根語

覚え方のコツ

覚え方

「あたら＝ああ、もったいない！」 副詞「あたら」で覚える → 「あたら若い命を…」＝「惜しくも若い命を…」

ポイント: 「あたらし」と「あらたし」の1文字違い - あたらし → た（惜しい） - あらたし → ら（新しい） 「た」と「ら」を入れ替えると意味が全く変わる！

関連する慣用表現・連語

表現 意味 入試重要度
あたらしき人 惜しいほど立派な人 ★★★
あたら (副詞) 惜しいことに ★★★
あたら口惜し 惜しくも残念だ ★★☆

暗記用まとめ

一言で

「あたらし」＝惜しい・もったいない。「新しい」は「あらたし」。

最重要: 古文「あたらし」≠現代語「新しい」。「惜しい」と訳す。

あやし

基本データ

品詞: 形容詞・シク活用 活用: あやしく／あやしく／あやし／あやしき／あやしけれ／○ 語源: 「あや（文・綾）」＋「し」＝理由のわからないこと。不思議・奇妙が原義。

辞書的意味

番号 意味 典型的な例文 出典
❶ 不思議だ、奇妙だ あやしきことかな 竹取物語
❷ 身分が低い、卑しい あやしき者 源氏物語
❸ 粗末だ、見苦しい あやしき住まひ 方丈記
❹ （感動して）不思議なほどだ、格別だ あやしうめでたし 源氏物語

派生関係：

原義：理由がわからない → 不思議だ
  ├→ ❶ 理解できない → 不思議だ・奇妙だ
  ├→ ❷ 通常でない → 身分が低い
  ├→ ❸ 通常でない → 粗末だ・見苦しい
  └→ ❹ 不思議なほど → 格別に（副詞的）
現代語との違い

現代語「怪しい」＝ 疑わしい、不審だ 古語「あやし」＝ 不思議だ ＋ 身分が低い ＋ 粗末だ 現代語にはない❷❸の意味が古文では非常に重要。

判別基準

意味 文脈上の手がかり 共起しやすい語
❶不思議だ 原因不明の出来事 「こと」「かな」「と思ふ」
❷身分が低い 人物の描写 「者」「人」「翁」
❸粗末だ 物・住まいの描写 「家」「住まひ」「衣」
❹格別だ 「あやしう」の副詞的用法 「めでたし」「をかし」「うつくし」
判別のコツ

人に使う→❷「身分が低い」。物に使う→❸「粗末だ」。 出来事に使う→❶「不思議だ」。副詞的に使う→❹「格別に」。

類義語・対義語

類義語 共通点 相違点
いやし（卑し） 「身分が低い」で❷と共通 「いやし」は身分の低さに特化
けしからず 「奇妙だ」で❶と近い 「けしからず」は不都合の意味も
対義語 意味
やむごとなし 高貴だ。❷の反対

入試での注意点

最重要

「あやしき者」＝「身分の低い者」が最頻出。「怪しい者」と訳すと不正解。 ❷❸の意味は現代語にないため、意識しないと見落とす。

  • 「あやしの民」＝身分の低い庶民（❷）
  • 「あやしき小家」＝粗末な小さい家（❸）
  • 「あやしう美しげなる」＝不思議なほど美しい（❹）
  • 竹取物語冒頭「あやしがりて」＝不思議に思って（❶）

覚え方のコツ

覚え方

3パターンを「人・物・事」で分ける: - 人＝身分が低い（❷）「あやしき者」 - 物＝粗末だ（❸）「あやしき家」 - 事＝不思議だ（❶）「あやしきこと」

ポイント: 「あやしき者」が出たら反射的に「身分の低い者」 → 竹取の「あやしがる」と源氏の「あやしき者」をペアで覚える → 「不思議 or 卑しい」の二択を常に意識

関連する慣用表現・連語

表現 意味 入試重要度
あやしき者 身分の低い者 ★★★
あやしがる 不思議に思う ★★★
あやしう＋形容詞 不思議なほど～ ★★☆
あやしの民 卑しい庶民 ★★☆

暗記用まとめ

一言で

「あやし」＝不思議だ＋身分が低い＋粗末だ。3義を使い分ける。

最重要: 「あやしき者」＝身分の低い者（≠怪しい者）。人→❷、物→❸、事→❶。

ありがたし

基本データ

品詞: 形容詞・ク活用 活用: ありがたく／ありがたく／ありがたし／ありがたき／ありがたけれ／○ 語源: 「有り＋難し」＝存在することが難しい → めったにない。

辞書的意味

番号 意味 典型的な例文 出典
❶ めったにない、珍しい ありがたきもの 枕草子
❷ （めったにないほど）立派だ、すばらしい ありがたき人 源氏物語
❸ 生きていくのが難しい、つらい 世はありがたし 方丈記

派生関係：

原義：有り＋難し ＝ 存在することが難しい
  ├→ ❶ めったに存在しない → 珍しい
  ├→ ❷ 珍しいほど優れている → 立派だ
  └→ ❸ 生きていることが難しい → つらい
      → 後世：珍しい → ありがたい → 感謝（現代語）
現代語との決定的な違い

現代語「ありがたい」＝ 感謝する 古語「ありがたし」＝ めったにない、珍しい 「珍しい」→「うれしい」→「感謝する」と意味がずれた。

判別基準

意味 文脈上の手がかり 共起しやすい語
❶めったにない 希少性の表現 「もの」「こと」「世に」
❷立派だ 人物評 「人」「心」「御」
❸つらい 生活苦の文脈 「世」「身」「命」

類義語・対義語

類義語 共通点 相違点
めでたし 「すばらしい」で❷と共通 「めでたし」は感嘆の対象
めづらし（珍し） 「珍しい」で❶と共通 「めづらし」は新鮮さへの賞賛も
対義語 意味
やすし（易し） 容易だ（「難し」の反対）

入試での注意点

最重要

「ありがたし」を「ありがたい（感謝）」と訳すのは不正解。 「めったにない」「珍しい」が古文での基本訳。

  • 枕草子「ありがたきもの」段＝「めったにないもの」を列挙
  • 「ありがたき御心」＝めったにないほど素晴らしい御心（❷）
  • 「世のありがたきを」＝世の中の暮らしにくさを（❸）

覚え方のコツ

覚え方

語源分解で即答: 「有り＋難し」＝有ることが難しい＝めったにない

ポイント: 枕草子の「ありがたきもの」段をセットで覚える → 「舅にほめらるる婿」「仕ふる人のよき」等、「めったにないもの」を列挙した段 → この段を知っていれば「ありがたし＝めったにない」は一生忘れない

意味の変遷: めったにない → うれしい → 感謝 の3段階変化

関連する慣用表現・連語

表現 意味 入試重要度
ありがたきもの めったにないもの ★★★
ありがたき人 めったにいないほど立派な人 ★★★
世はありがたし 世の中は暮らしにくい ★★☆

暗記用まとめ

一言で

「ありがたし」＝「有り＋難し」＝めったにない。感謝の意味は後世の派生。

最重要: 古文「ありがたし」≠現代語「ありがたい」。「めったにない」と訳す。

いたづらなり

基本データ

品詞: 形容動詞・ナリ活用 活用: いたづらなら／いたづらなり・に／いたづらなり／いたづらなる／いたづらなれ／いたづらなれ 語源: 「いた（甚）＋つら（連）」が訛ったとも。むだ・空虚が原義。

入試解説

京大2024での詳細分析は 入試解説/いたづらなり を参照

辞書的意味

番号 意味 典型的な例文 出典
❶ むだだ、無益だ いたづらに心を動かす 古今和歌集仮名序
❷ つまらない、退屈だ いたづらに日を過ぐす 徒然草
❸ 暇だ、することがない いたづらに暇ありげなる 源氏物語
❹ むなしい、はかない いたづらに立てり 徒然草
❺ 死ぬ（「いたづらになる」の形で） いたづらになりはてたまふ 源氏物語

派生関係：

原義：むだだ、役に立たない
  ├→ ❶ 無益だ、甲斐がない
  ├→ ❷ つまらない、退屈だ
  ├→ ❸ 暇だ → やることがない
  ├→ ❹ むなしい、空虚だ
  └→ ❺ 空しくなる → 死ぬ（婉曲表現）
現代語との決定的な違い

現代語「いたずら」＝ 悪戯、悪ふざけ 古語「いたづらなり」＝ むだだ、暇だ、むなしい 「いたずらっ子」のような意味は古文にはない。

判別基準

意味 文脈上の手がかり 共起しやすい語
❶むだだ 行為の無益さ 「心」「言の葉」「涙」
❷つまらない 感情の停滞 「日」「ころ」
❸暇だ 時間を持て余す 「暇」「つれづれ」
❹むなしい 虚しさ・はかなさ 「立つ」「ある」
❺死ぬ 生死の文脈 「なる」「なりはつ」

類義語・対義語

類義語 共通点 相違点
つれづれなり 「退屈だ」で❷と共通 「つれづれ」はしみじみとした退屈
はかなし 「むなしい」で❹と共通 「はかなし」ははかない・頼りない
むなし（空し） 「空虚だ」で❹と共通 ほぼ同義

入試での注意点

最重要

「いたづらなり」を「いたずらだ」と訳すのは完全な誤り。 「いたづらになる」＝「死ぬ」は超頻出の婉曲表現。

  • 小野小町「いたづらにわが身世にふる」＝むなしく過ごす（❶❹）
  • 「いたづらに立てりけり」＝むなしく（役立たず）立っていた（❹）
  • 源氏物語で「いたづらになる」は死の婉曲表現として繰り返し出る

覚え方のコツ

覚え方

「いたづら＝むだ」で覚える。悪戯は完全に忘れる。

決め手の例文: 徒然草の水車の話 → 水車が「いたづらに立てりけり」＝むだに立っていた（回らない水車） → いたずら（悪戯）をする水車ではない！

ポイント: 「いたづらになる」＝死ぬ は別格で覚える → 源氏物語で繰り返し出る → 空しくなる ＝ この世からなくなる ＝ 死ぬ

関連する慣用表現・連語

表現 意味 入試重要度
いたづらになる 死ぬ ★★★
いたづらに過ぐ むだに時を過ごす ★★★
いたづらに暮らす することなく暮らす ★★☆

暗記用まとめ

一言で

「いたづらなり」＝むだ・暇・むなしい。「いたづらになる」＝死ぬ。

最重要: 古文「いたづら」≠現代語「いたずら（悪戯）」。「むだ」「むなしい」と訳す。

いとほし

基本データ

品詞: 形容詞・シク活用 活用: いとほしく／いとほしく／いとほし／いとほしき／いとほしけれ／○ 語源: 「いと（甚）＋ほし」＝甚だしく思われる。相手の苦境に心が痛む。

辞書的意味

番号 意味 典型的な例文 出典
❶ 気の毒だ、かわいそうだ いとほしき人 源氏物語
❷ 困った、つらい、きまり悪い いとほしきこと 枕草子
❸ かわいい、いじらしい いとほしき御さま 源氏物語

派生関係：

原義：見ていて心が痛む
  ├→ ❶ 相手の苦境に同情 → 気の毒だ（最重要）
  ├→ ❷ 自分の状況がつらい → 困った・きまり悪い
  └→ ❸ 弱く小さいものへの情 → かわいい（近世以降中心）
      → 現代語: かわいい → 愛おしい
現代語との違い

現代語「いとおしい」＝ 愛おしい、大切に思う 古語「いとほし」＝ 気の毒だ、かわいそうだ（最重要義） 古文では❶「同情」が中心。❸「かわいい」は近世以降に増加。

判別基準

意味 文脈上の手がかり 共起しやすい語
❶気の毒だ 他者への同情 「人」「さま」「御」
❷困った 自分の窮状 「こと」「身」
❸かわいい 幼い・弱いものへの情 「御さま」「幼き」

類義語・対義語

類義語 共通点 相違点
かなし 「かわいい」で❸と共通 「かなし」は愛しさの切実さが強い
あはれなり 「同情」で❶と共通 「あはれ」はより広いしみじみとした感動

入試での注意点

最重要

「いとほし」は❶「気の毒だ」が最頻出。「かわいい」「愛おしい」と安易に訳さない。 主語が「見る側」なら❶❸、「見られる側」への視点なら❷。

  • 「いとほしく思す」＝気の毒にお思いになる（❶＋尊敬）
  • 源氏物語で女君の窮状に対して「いとほし」と多用される
  • 「いとほしげなり」＝気の毒そうだ / かわいらしげだ

覚え方のコツ

覚え方

「いとほし＝気の毒」が鉄則。愛おしいは後の意味。

語源で覚える: 「いと（甚だしく）＋ほし（思われる）」 → 見ていて甚だしく心が痛む → 気の毒だ

ポイント: 源氏物語で光源氏が女性に対して「いとほし」と感じる場面 → ほぼ全て「気の毒だ」の文脈（不遇な女性への同情） → 「かわいい」で訳すと文脈がおかしくなるケースが多い

関連する慣用表現・連語

表現 意味 入試重要度
いとほしく思す 気の毒にお思いになる ★★★
いとほしげなり 気の毒そうだ ★★☆
いとほしき人 気の毒な人 ★★☆

暗記用まとめ

一言で

「いとほし」＝気の毒だ（最重要）＋困った＋かわいい。

最重要: 古文「いとほし」≠現代語「いとおしい」。「気の毒だ」が基本訳。

いのる（祈る）

基本データ

品詞: 動詞・ラ行四段 活用: いのら／いのり／いのる／いのる／いのれ／いのれ 語源: 「い（斎）＋のる（宣る・告る）」＝神仏に言葉を申し述べる。

辞書的意味

番号 意味 典型的な例文 出典
❶ 祈る、祈願する 仏にいのる 源氏物語
❷ 呪う、呪詛する 人をいのる 源氏物語

派生関係：

原義：神仏に強く訴える
  ├→ ❶ よいことを願う → 祈願する
  └→ ❷ 悪いことを願う → 呪詛する
現代語との決定的な違い

現代語「祈る」＝ よいことを願う（ポジティブのみ） 古語「いのる」＝ 祈願する ＋ 呪詛する（ネガティブも！） 古文では❷「呪う」が非常に重要。源氏物語で多用される。

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶祈願する 病気平癒・安産・出世を願う場面。寺社での祈祷の場面。 「仏」「神」「加持」「祈祷」
❷呪詛する 恨みのある相手を害そうとする場面。物の怪・怨霊の文脈。 「人を」「もののけ」「調伏」
文脈で区別するポイント

「仏に」いのる → ❶ 祈願（神仏が対象＝よい願い） 「人を」いのる → ❷ 呪詛（人が対象＝害を願う） 格助詞「に」と「を」の違いが最大の手がかり。

用例（意味番号付き）

意味 用例 文脈解説
❶祈願 「今日始むべき祈禱ども、さるべき人々うけたまはれる」（源氏物語） 病気平癒のために僧侶が祈祷する場面
❷呪詛 「人をいのりて害せむとする」（源氏物語） 六条御息所の生霊が葵の上を呪詛する文脈

類義語・対義語

類義語 共通点 相違点
のろふ（呪ふ） 「呪う」で❷と共通 「のろふ」は呪いに特化
ねんず（念ず） 「祈る」で❶と共通 「ねんず」は心の中で念じる

入試での注意点

最重要

「いのる」に❷「呪う」の意味があることを知らないと、源氏物語の物の怪関連の文が全く読めない。 六条御息所の生霊エピソードは超頻出。

  • 源氏物語で「いのる」は祈祷と呪詛の両方で繰り返し出現
  • 「いのり」（名詞）＝祈祷 or 呪詛
  • 「調伏（ちょうぶく）」＝物の怪を退治する祈祷 → ❶の文脈

覚え方のコツ

覚え方

「いのる＝祈る＋呪る」 プラスとマイナス両方ある

語源で覚える: 神仏に「強く訴える」が原義 → よい願い ＝ 祈る（❶） → 悪い願い ＝ 呪う（❷）

おろかなり

基本データ

品詞: 形容動詞・ナリ活用 活用: おろかなら／おろかなり・に／おろかなり／おろかなる／おろかなれ／おろかなれ 語源: 「おろ（疎）＋か」＝まばらだ、いいかげんだ。配慮が行き届かない。

辞書的意味

番号 意味 典型的な例文 出典
❶ いいかげんだ、おろそかだ おろかに思ふ 源氏物語
❷ 並ひととおりだ、普通だ おろかならず 源氏物語
❸ 愚かだ、道理がわからない おろかなる人 徒然草

派生関係：

原義：まばらだ、行き届かない
  ├→ ❶ 配慮が行き届かない → いいかげんだ（最重要義）
  ├→ ❷ 並大抵の程度 → 普通だ
  └→ ❸ 知恵が行き届かない → 愚かだ（現代語の意味）
現代語との決定的な違い

現代語「愚か」＝ 頭が悪い、知性が劣る 古語「おろかなり」＝ いいかげんだ、おろそかだ（最重要義） 古文での❸「愚か」の用法は少数派。❶❷が圧倒的に多い。

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶いいかげん 態度・行為がおろそかな場面。心がこもっていない、手を抜いている。 「思ふ」「もてなす」「言ふ」
❷並大抵 否定形「おろかならず」で「並大抵ではない＝大変だ」の意味。 「ならず」「ならぬ」
❸愚かだ 知性・判断力を評価する場面。現代語と同じ文脈。 「人」「心」
最重要パターン

「おろかならず」＝並大抵でない＝大変だ・格別だ この否定形が入試で最も出る。「おろかならず悲し」＝並大抵でなく悲しい。

用例（意味番号付き）

意味 用例 文脈解説
❶いいかげん 「おろかに思さるるにやあらむ」（源氏物語） いいかげんにお思いになるのだろうか→愛情不足の懸念
❷並大抵でない 「おろかならず思す」（源氏物語） 並大抵でなくお思いになる→深い愛情
❸愚かだ 「おろかなる人の目にも」（徒然草） 愚かな人の目にも→知性の評価

類義語・対義語

類義語 共通点 相違点
なほざりなり 「いいかげん」で❶と共通 ほぼ同義
あだなり 「誠実でない」で❶と近い 「あだ」は浮気・不実

入試での注意点

最重要

「おろかなり」を「愚かだ」と訳すのは多くの場合不正解。 「おろかならず」＝並大抵でないを即答できるかが勝負。

  • 「おろかならず思ふ」＝大変深く思う（❷の否定＝格別だ）
  • 「おろかに言ふ」＝いいかげんに言う（❶）
  • 「おろかなり」が「愚かだ」（❸）になるのは知性を直接評価する文脈のみ

覚え方のコツ

覚え方

「おろか ＝ おろそか」 → いいかげん → 「おろか」と「おろそか」は同根語！

決め手: 「おろかならず」＝ 並大抵でない ＝ すごい → 否定形が来たら「格別だ、大変だ」と訳す

ポイント: 頭の悪さ（現代語の「愚か」）は古文では少数派 → まず「いいかげん」で訳してみる → 通じなければ「愚か」を検討

表現 意味 入試重要度

おろかに思ふ いいかげんに思う ★★☆

暗記用まとめ

一言で

「おろかなり」＝いいかげん・おろそか。「おろかならず」＝格別だ。

かたし

基本データ
  • 難し（かたし）: ク活用 ＝ 困難だ
  • 堅し（かたし）: ク活用 ＝ 堅い、しっかりしている 活用: かたく／かたく／かたし／かたき／かたけれ／○ 注意: 多義語カテゴリにも 多義語/かたし として登録あり。

辞書的意味

難し〔ク活用〕

番号 意味 典型的な例文 出典
❶ 難しい、困難だ 世にふるはかたきものかな 小野小町
❷ めったにない、まれだ かたき人 源氏物語

堅し〔ク活用〕

番号 意味 典型的な例文 出典
❸ 堅い、しっかりしている かたき心 源氏物語
❹ 誠実だ、律義だ かたき人 大鏡
現代語との違い

現代語「固い/硬い/堅い」＝ 物理的な硬度が中心 古語「かたし」＝ 難しい（❶）・めったにない（❷）が重要 ❶❷の「難し」系は現代語の「固い」と全く別の意味。

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶難しい 実現・達成が困難な場面。恋の成就や世渡りの難しさ。 「世」「こと」「ふる」
❷めったにない 希少性を表す場面。「ありがたし」と同義になる。 「人」「もの」
❸堅い 物事がしっかりしている場面。意志・態度の堅さ。 「心」「門」「約束」
❹誠実だ 人柄の評価。浮気しない、きちんとしている。 「人」「女」
「かたき人」の二つの意味
  • 難しの文脈 → ❷「めったにいない人」
  • 堅しの文脈 → ❹「誠実な人、堅物な人」 → 恋愛文脈なら❹（なびかない人）、賞賛なら❷（稀な人）

用例（意味番号付き）

意味 用例 文脈解説
❶困難 「世にふるはかたきものかな」（小野小町） 世の中で生きていくのは難しいことだ→人生の困難
❷稀だ 「かたき人の御ありさま」（源氏物語） めったにない方のご様子→稀有な賞賛
❹誠実 「いとかたき人にて」（源氏物語） とても誠実な（堅い）人で→人柄の評価

類義語・対義語

類義語 共通点 相違点
ありがたし 「めったにない」で❷と共通 「ありがたし」の方が頻出
対義語 意味
やすし（易し） 容易だ。❶の反対
あだなり 不誠実だ。❹の反対

入試での注意点

最重要

「かたし」は「難し」と「堅し」の2系統。 「かたき人」が❷か❹かを文脈で判断する問題が出る。

覚え方のコツ

覚え方

2系統を分ける: 「難」＝困難 / 「堅」＝しっかり

ポイント: 恋愛文脈の「かたき人」＝❹「堅物、なびかない人」 → 「かたくなな人」のイメージ 賞賛文脈の「かたき人」＝❷「めったにいない人」 → 「ありがたし」と同義

関連する慣用表現・連語

表現 意味 入試重要度
かたき人 めったにいない人 / 誠実な人 ★★★
世にふるはかたし 世を生きるのは難しい ★★☆
かたき心 堅い意志 ★★☆

暗記用まとめ

一言で

「かたし」＝難し（困難・稀）＋堅し（堅い・誠実）の2系統。

最重要: 「かたき人」は文脈で「稀な人」か「堅物」か判断。

かなし

基本データ
  • 愛し（かなし）: シク活用 ＝ かわいい、いとおしい
  • 悲し・哀し（かなし）: シク活用 ＝ 悲しい、哀しい 活用: かなしく／かなしく／かなし／かなしき／かなしけれ／○ 注意: 入試解説は 入試解説/かなし 、基本解説は 重要形容詞/かなし も参照。

辞書的意味

愛し〔シク活用〕

番号 意味 典型的な例文 出典
❶ かわいい、いとおしい いとかなしうしたまふ 伊勢物語
❷ 心にしみる、身にしみて感じる かなしき秋の夕暮れ 古今和歌集

悲し・哀し〔シク活用〕

番号 意味 典型的な例文 出典
❸ 悲しい、つらい かなしきことかぎりなし 竹取物語
❹ かわいそうだ、不憫だ かなしき人 源氏物語
現代語との決定的な違い

現代語「悲しい」＝ 悲嘆（❸のみ） 古語「かなし」＝ かわいい（❶）が最重要義 古文で「かなし」が出たら、まず❶「かわいい」を検討する。

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶かわいい 親子・男女間の愛情。幼い子や恋人への深い愛しさ。「いと」と共起して強調されることが多い。 「いと」「子」「姫」「したまふ」
❷身にしみる 自然美や風情に心動かされる場面。秋の夕暮れなど、しみじみとした感動。 「秋」「夕暮れ」「月」
❸悲しい 別離・死・不幸の場面。現代語と同じ「悲嘆」の文脈。 「泣く」「涙」「別れ」
❹かわいそう 他者の不幸に同情する場面。見ていて不憫に思う。 「人」「さま」
❶と❸の見分け方

「いとかなし」＋愛情の文脈 → ❶ かわいい 「かなし」＋別離・死の文脈 → ❸ 悲しい 迷ったら周囲の語（泣く→❸、したまふ→❶）で判断。

用例（意味番号付き）

意味 用例 文脈解説
❶かわいい 「ひとつ子にさへありければ、いとかなしうしたまひけり」（伊勢物語） 一人っ子なので大変かわいがった→親の愛情
❷身にしみる 「秋はかなしきもの」（枕草子） 秋は心にしみるもの→季節の情趣
❸悲しい 「泣き惑ふことかぎりなし。かなしきこと」（竹取物語） 泣き乱れること限りない。悲しいこと→かぐや姫の別れ

類義語・対義語

類義語 共通点 相違点
うつくし 「かわいい」で❶と共通 「うつくし」は小さいものへの愛着、「かなし」は切実な愛情
いとほし 「かわいそう」で❹と共通 「いとほし」は気の毒、「かなし」は不憫
あはれなり 「しみじみ」で❷と共通 「あはれ」はより広い感動

入試での注意点

最重要

「かなし」に❶「かわいい」の意味があることを見落とすと大量失点。 「いとかなしうしたまふ」＝「とてもかわいがりなさる」が即答できるか。

  • 「かなし」の漢字表記が「愛」なら❶❷、「悲・哀」なら❸❹
  • ただし古文では漢字がないことが多いので文脈で判断
  • 和歌で「かなし」は❷「身にしみる」＋❸「悲しい」の両義に取れることもある

覚え方のコツ

覚え方

「かなし＝かわいい」が古文の基本！悲しいは二番手。

漢字で覚える: 「愛し」＝かわいい、「悲し」＝悲しい → 「愛」の字を当てるのが古文の中心義

ポイント: 「いとかなし」＋人物 ＝ 「かわいい」 → 「いと悲し」なら別離・死の場面のはず → 愛情場面で「かなし」が出たら「かわいい」一択

関連する慣用表現・連語

表現 意味 入試重要度
いとかなしうしたまふ とてもかわいがりなさる ★★★
かなしきこと 悲しいこと / 愛しいこと ★★★
身にしみてかなし 身にしみて心が動く ★★☆

暗記用まとめ

一言で

「かなし」＝愛し（かわいい）＋悲し（悲しい）の2系統。

最重要: 古文「かなし」の中心義は「かわいい」。「悲しい」だけではない。

くちをし（口惜し）

基本データ

品詞: 形容詞・シク活用 活用: くちをしく／くちをしく／くちをし／くちをしき／くちをしけれ／○ 語源: 「朽ち＋惜し」＝せっかくのものが台無しになって惜しい → 残念だ。

辞書的意味

番号 意味 典型的な例文 出典
❶ 残念だ、がっかりだ くちをしきこと 枕草子
❷ つまらない、取るに足りない くちをしき身 源氏物語
❸ 悔しい くちをしう思ふ 大鏡

派生関係：

原義：期待していたものが台無しで惜しい
  ├→ ❶ 期待はずれ → 残念だ、がっかりだ（最重要義）
  ├→ ❷ 価値が低い → つまらない、取るに足りない
  └→ ❸ 悔やまれる → 悔しい
      → 現代語:「口惜しい」＝悔しい（❸に限定）
現代語との決定的な違い

現代語「口惜しい/くやしい」＝ 悔しい（怒りのニュアンス） 古語「くちをし」＝ 残念だ、がっかりだ（落胆のニュアンス） 古文では❶「残念、がっかり」が中心。怒りよりも失望の感情。

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶残念だ 期待していたことが実現しなかった場面。花が散った、人に会えなかった等。 「こと」「思ふ」「見る」
❷つまらない 自分や物事の価値を低く評価する場面。謙遜・自嘲の文脈。 「身」「もの」「人」
❸悔しい 取り返しのつかないことを悔やむ場面。機会を逃した後悔。 「思ふ」「覚ゆ」
判別のコツ

状況への落胆→❶、自己評価→❷、後悔→❸。 迷ったら❶「残念だ」で訳す。「怒り」の意味は薄い。

用例（意味番号付き）

意味 用例 文脈解説
❶残念 「花の散るを、くちをしと思ふ」（枕草子） 花が散るのを残念に思う→期待はずれへの落胆
❷つまらない 「くちをしき身なれば」（源氏物語） つまらない身分なので→自己の低評価
❸悔しい 「くちをしう思して」（大鏡） 悔しくお思いになって→機会喪失への後悔

類義語・対義語

類義語 共通点 相違点
あたらし 「惜しい」で❶と共通 「あたらし」は価値ある物の無駄を惜しむ
ねたし 「悔しい」で❸と近い 「ねたし」は嫉妬の感情が強い
対義語 意味
おもしろし 趣がある（期待以上の満足）

入試での注意点

最重要

「くちをし」は「悔しい」（怒り）ではなく「残念だ」（落胆）がメイン。 「くちをしきこと」＝残念なこと（❶）が最頻出。

  • 「くちをし」と「ねたし」の区別を問う問題がある
  • くちをし＝自分の期待外れへの失望
  • ねたし＝他人への嫉妬
  • 「くちをし」と「あたらし」の区別も重要
  • くちをし＝期待が実現しない失望
  • あたらし＝優れたものが無駄になる惜しさ

覚え方のコツ

覚え方

語源: 「朽ち惜し」＝ 朽ちてしまって惜しい → せっかくのものが台無し → 残念！がっかり！

ポイント: 「悔しい」より「残念」のニュアンス → 怒っているのではなく、しょんぼりしている感じ

対義語「おもしろし」とセットで: おもしろし ＝ 心晴れる ↔ くちをし ＝ がっかり

一言で

けしき（気色）

基本データ

品詞: 名詞 語源: 「気色」＝気（き）の色 → 外に現れた様子・態度。

辞書的意味

番号 意味 典型的な例文 出典
❶ 様子、態度 けしきあり 源氏物語
❷ 表情、顔つき けしきの変はる 枕草子
❸ 兆し、きざし 雨のけしき 徒然草
❹ 意向、気持ち けしきを見る 大鏡

派生関係：

原義：内面が外に表れたもの
  ├→ ❶ 人の態度・振る舞い
  ├→ ❷ 顔に現れた表情
  ├→ ❸ 自然に現れた兆し → 天候の前兆
  └→ ❹ 心中が態度に表れたもの → 意向
      → 現代語:「景色」＝風景の意味に変化
現代語との決定的な違い

現代語「景色」＝ 風景、自然の眺め 古語「けしき」＝ 様子、態度、表情、意向 古文で「けしき」は「風景」の意味ではほぼ使わない。

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶様子 人の振る舞い・場の雰囲気を観察する場面。「けしきあり」は「そぶりがある」。 「あり」「なし」「見ゆ」
❷表情 顔色・顔つきの変化。感情が顔に出る場面。 「変はる」「顔」「目」
❸兆し 天候・状況の前兆。雨・風・季節の変化の気配。 「雨」「風」「空」
❹意向 人の本心を探る場面。「けしきを見る」＝意向を探る。 「見る」「うかがふ」「知る」
判別のコツ

人の態度→❶❷、自然→❸、心中→❹ 「けしきを見る」＝意向を探る（❹）が最頻出パターン。

用例（意味番号付き）

意味 用例 文脈解説
❶様子 「帝のけしき、いとあはれに心苦しく」（源氏物語） 帝のご様子がとてもしみじみと痛々しく→態度の観察
❸兆し 「雨のけしきなり」（徒然草） 雨の兆しだ→天候の前兆
❹意向 「けしきを見て」（大鏡） 意向を探って→人の本心を読む

類義語・対義語

類義語 共通点 相違点
さま（様） 「様子」で❶と共通 「さま」は外見全般、「けしき」は内面の表出
ありさま 「様子」で❶と共通 「ありさま」は状態全般

入試での注意点

最重要

「けしき」を「景色（風景）」と訳すのは不正解。 「様子」「態度」「意向」のいずれかで訳す。

  • 「御けしき」＝お気持ち、ご様子（❶❹＋尊敬）
  • 「けしきばむ」＝それらしい様子をする、気取る
  • 「けしきを見る」＝意向を探る（❹）→ 政治・恋愛場面で超頻出

覚え方のコツ

覚え方

「気色（けしき）」＝ 気の色 ＝ 内面が外に出た色 → 態度・表情・意向 → 「風景」は関係ない！

決め手の表現: 「けしきを見る」＝ 意向を探る → 「景色を見る」ではない！ → 「あの人の気持ちを探る」

ポイント: 「景色」（風景）は近世以降の意味変化 → 古文では「けしき＝心の表れ」と覚える

辞書的意味

番号 意味 典型的な例文 出典

判別のコツ

❶❷はほぼ同じ意味。常に最上級の褒め言葉として使われる。 「さうなし」が出たら「この上ない」と訳せばまず正解。

意味 用例 文脈解説

❷すばらしい 「さうなくめでたき御ありさま」（栄花物語） 比類なくすばらしいお姿→人物の絶賛

最重要

  • 「さうなき人」＝並ぶ者のない（最高の）人

覚え方

基本データ

  • 忍ぶ（しのぶ）: バ行四段（他動詞）＝ 我慢する、堪える
  • 偲ぶ（しのぶ）: バ行四段 ＝ 懐かしむ、思い出す 注意: 漢字表記が同じ「忍ぶ」でも上二段と四段で意味が異なる。

辞書的意味

忍ぶ〔バ行上二段・自動詞〕

番号 意味 典型的な例文 出典
❶ 人目を避ける、隠れる しのびて渡る 建礼門院右京大夫集

忍ぶ〔バ行四段・他動詞〕

番号 意味 典型的な例文 出典
❸ 我慢する、堪える しのぶるになむありける 大和物語

番号 意味 典型的な例文 出典

❺ 故人を偲ぶ、追慕する 故人をしのぶ 源氏物語
現代語との決定的な違い

現代語「忍ぶ」＝ 我慢する、隠れる（❶❸のみ） 古語「しのぶ」＝ ❹❺「懐かしむ・偲ぶ」が入試最重要 現代語にはない「偲ぶ」系の意味を見落とすと失点する。

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶隠れる 人目を避けて行動する場面。密会・秘密の外出。 「て」「渡る」「出づ」
❷こっそり ❶の連用形的用法。「しのびて＋動詞」で「こっそり～する」。 「通ふ」「参る」「語る」
❸我慢する 感情を抑える場面。悲しみ・怒り・嫉妬を堪える。 「涙」「心」「思ひ」
❹懐かしむ 過去の思い出を振り返る場面。昔の恋・季節・場所。 「昔」「むかし」「ころ」
❺追慕する 亡くなった人を思い出す場面。故人への追悼。 「人」「故人」「名残」
判別のコツ

「しのびて」（副詞的）→ ❶❷ こっそり 「〜をしのぶ」（他動詞）→ ❸我慢 or ❹❺偲ぶ → 目的語が感情なら❸、過去・故人なら❹❺

用例（意味番号付き）

意味 用例 文脈解説

❸我慢する 「心地にはかぎりなくねたく心憂う思ふを、しのぶるになむありける」（大和物語） この上なく妬ましくつらく思うのを、こらえていたのだった→感情の抑制
❹懐かしむ 「浅茅が宿に昔しのぶこそ、色好むとは言はめ」（徒然草） 荒れ果てた家で昔を懐かしむのこそ、風流と言えよう→過去への追想
❺追慕 「（亡き資盛を）しのびあまりて」（建礼門院右京大夫集） （亡くなった資盛を）偲ぶあまりに→故人への追悼

恋ふ 「思い慕う」で❹❺と近い 「恋ふ」は現在の対象への思慕、「しのぶ」は過去を振り返る

最重要

  • 「しのびて」＝こっそり（❶❷）。副詞的に使う。
  • 「〜をしのぶ」の目的語で判断: 感情→❸、昔・人→❹❺
  • 和歌では「しのぶ」と植物「忍草（しのぶぐさ）」の掛詞になることがある
  • 伊勢物語冒頭「しのぶずりの狩衣」の「しのぶ」は植物名（別語）

覚え方

決め手: 「昔をしのぶ」は「偲ぶ」。「忍者」ではない。

ポイント: 現代語でも「偲ぶ」は使うが、古文ではもっと頻出

派生関係：

原義：物事について語ること
  ├→ ❶ 人と語り合う → おしゃべり、世間話（最重要義）

  └→ ❸ 文学ジャンル → 物語作品

❸物語作品 文学作品名や読書の場面。竹取・源氏等の作品を指す。 「読む」「書く」「竹取の」

基本データ

番号 意味 典型的な例文 出典

現代語では「さうなし」は使わない（死語） 古語「さうなし」＝ 並ぶものがない、最高だ（最上級の褒め言葉） 現代語に直接対応する語がないため、意味を知らないと訳せない。

意味 どんな文脈で？ 共起しやすい語
❶すばらしい 人物の才能・地位・人柄を賞賛する場面。最高級の褒め言葉。 「人」「御さま」「御かたち」
❷美しい 容姿・風景・物の美しさに感嘆する場面。視覚的な素晴らしさ。 「花」「色」「さま」
❸喜ばしい 出産・昇進・婚姻などの慶事。お祝いの場面。 「こと」「よろこぶ」
判別のコツ

人物評→❶、外見→❷、慶事→❸。 古文では❶が最も多い。「めでたし＝すばらしい」がデフォルト。

用例（意味番号付き）

❷美しい 「花のいとめでたく咲きたる」（枕草子） 花がとても見事に咲いている→視覚的な美への感嘆
❸喜ばしい 「めでたきことかぎりなし」（大鏡） 喜ばしいこと限りない→慶事の賀辞

入試での注意点

最重要

「めでたし」を「めでたい（お祝い）」だけで訳すのは不十分。 「すばらしい」「立派だ」が古文での基本訳。

  • 「めでたき人」＝すばらしい人（❶）→「めでたい人」ではない
  • 語源「愛で＋甚し」を知っていると全義が理解できる
  • 「いとめでたし」＝とてもすばらしい が最頻出パターン

覚え方のコツ

覚え方

語源: 「愛で甚し」＝ 感嘆するほどだ → すばらしい！ → 「めでる」（愛でる・感嘆する）＋「いたし」（甚だしい）

ポイント: 「めでたし＝すばらしい」。お祝いの意味は二番手。

もてなす（もて成す）

基本データ

品詞: 動詞・サ行四段 活用: もてなさ／もてなし／もてなす／もてなす／もてなせ／もてなせ 語源: 「持て（もて）＋成す」＝手に取って処置する → 取り扱う、振る舞う。

辞書的意味

番号 意味 典型的な例文 出典
❶ 取り扱う、待遇する さまざまにもてなす 源氏物語
❷ 振る舞う、ふりをする 知らず顔にもてなす 源氏物語
❸ とりつくろう、装う よろしうもてなす 枕草子
❹ 歓待する、ごちそうする いみじうもてなす 大鏡

派生関係：

原義：手に取って処置する → 取り扱う
  ├→ ❶ 人を処遇する → 取り扱う、待遇する（最重要義）
  ├→ ❷ 自分の態度を処置する → 振る舞う、ふりをする
  ├→ ❸ 外見を処置する → とりつくろう、装う
  └→ ❹ 客を処遇する → 歓待する
      → 現代語:「もてなす」＝❹ 歓待する（に限定）
現代語との決定的な違い

現代語「もてなす」＝ お客さんを歓待する（❹のみ） 古語「もてなす」＝ 取り扱う、振る舞う（❶❷が中心） 古文では「歓待」よりも「態度をとる」「ふりをする」の方がはるかに多い。

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶取り扱う 人を待遇する場面。良い扱い・悪い扱い両方。 「人」「さまざまに」「よく」
❷ふりをする 本心を隠して態度を装う場面。「～顔にもてなす」が典型。 「顔」「知らず」「さりげなく」
❸とりつくろう 外見や状況を取り繕う場面。体裁を整える。 「よろしう」「つくろふ」
❹歓待する 客をもてなす場面。食事・酒・贈り物。 「いみじう」「客」「饗応」
判別のコツ

「～顔にもてなす」→ ❷ ～のふりをする（最頻出パターン） 人を主語→ ❶❹、自分が主語→ ❷❸ 迷ったら❶「取り扱う」で訳してみる。

用例（意味番号付き）

意味 用例 文脈解説
❶取り扱う 「さまざまにもてなしたまふ」（源氏物語） さまざまにお取り扱いになる→人への待遇
❷ふりをする 「知らず顔にもてなしたまへど」（源氏物語） 知らない顔を装いなさるけれども→本心を隠す態度
❸とりつくろう 「よろしうもてなしたる」（枕草子） うまくとりつくろっている→体裁を整える
❹歓待 「いみじうもてなして」（大鏡） 大層もてなして→客への歓待

類義語・対義語

類義語 共通点 相違点

つくろふ 「とりつくろう」で❸と共通 「つくろふ」は修繕・整理も含む

入試での注意点

最重要

「もてなす」を「歓待する」とだけ訳すのは不十分。 「～顔にもてなす」＝～のふりをする（❷）が入試最頻出。

  • 「知らず顔にもてなす」＝知らないふりをする → 源氏物語で頻出
  • 「もてなし」（名詞形）＝態度、振る舞い、取り扱い
  • 「歓待する」の意味は文脈が明らかなときだけ

覚え方のコツ

覚え方

語源: 「持て成す」＝ 手に取って成す ＝ 処置する → 「お客をもてなす」は処置の一部にすぎない

決め手: 「～顔にもてなす」＝「～のふりをする」 → 知らず顔、平気顔、涼しい顔…＋もてなす

ポイント: 古文の「もてなす」は「態度をとる」。 「おもてなし」（歓待）は現代語の限定的な意味。

ものがたり（物語）

基本データ

品詞: 名詞（動詞「ものがたる」「ものがたりす」もあり） 語源: 「もの（物事）＋かたり（語り）」＝ 物事について語ること。

辞書的意味

番号 意味 典型的な例文 出典
❶ おしゃべり、世間話 ものがたりなどしてゐたるに 枕草子
❷ 話、語り伝えられたこと 昔のものがたり 徒然草
❸ 物語作品（文学ジャンル名） 竹取のものがたり 竹取物語

派生関係：

原義：物事について語ること
  ├→ ❶ 人と語り合う → おしゃべり、世間話（最重要義）
  ├→ ❷ 語り伝えられた話 → 話、言い伝え
  └→ ❸ 文学ジャンル → 物語作品
      → 現代語:「物語」＝❸ フィクション作品（に限定）
現代語との決定的な違い

現代語「物語」＝ フィクション作品、ストーリー（❸のみ） 古語「ものがたり」＝ おしゃべり、世間話（❶が最重要） 古文で「ものがたりす」は「おしゃべりをする」の意味がほとんど。

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶おしゃべり 人と会って話す場面。日常の会話・雑談・語り合い。 「す」「しあふ」「ゐたり」
❷話 過去の出来事や言い伝えを語る場面。昔話・エピソード。 「昔」「聞く」「語る」
❸物語作品 文学作品名や読書の場面。竹取・源氏等の作品を指す。 「読む」「書く」「竹取の」
判別のコツ

「ものがたりす」「ものがたりしあふ」→ ❶ おしゃべりする 作品名＋「ものがたり」→ ❸ 物語作品 人と人が会話している場面→❶、それ以外→❷❸。

用例（意味番号付き）

意味 用例 文脈解説
❶おしゃべり 「ものがたりなどしてゐたるに」（枕草子） おしゃべりなどしていると→日常の雑談場面
❶おしゃべり 「女房どもとものがたりしたまふ」（源氏物語） 女房たちとお話をなさる→会話・語らい
❸物語作品 「竹取のものがたり」（竹取物語） 竹取物語→文学作品名

類義語・対義語

類義語 共通点 相違点
かたらふ 「語り合う」で❶と近い 「かたらふ」は動詞、「ものがたり」は名詞
よばなし（夜話） 「おしゃべり」で❶と近い 「よばなし」は夜の語らいに限定

入試での注意点

最重要

「ものがたり」を「物語（小説）」とだけ訳すのは不十分。 「ものがたりす」＝おしゃべりをするが入試最頻出。

  • 「ものがたりす」は「す」がサ変動詞。「おしゃべりする」の意味。
  • 「ものがたりしあふ」＝語り合う、おしゃべりし合う
  • 作品名でない「ものがたり」は❶「おしゃべり」が基本

覚え方

決め手: 「ものがたりす」＝「おしゃべりする」 → 「物語する」ではおかしい → 「おしゃべりする」と訳す

ポイント: 古文の「ものがたり」は日常会話。フィクションではない。

もののけ（物の怪）

基本データ

品詞: 名詞 語源: 「もの（霊的存在）＋け（気）」＝ 霊の気配 → 人に取り憑く霊。 表記: 「物の怪」「もののけ」

辞書的意味

番号 意味 典型的な例文 出典
❶ 人に取り憑いて苦しめる霊 もののけにわづらふ 源氏物語
❷ 死霊・生霊 もののけのさまざまに出で来 源氏物語
❸ 霊の祟り、怪異現象 もののけなどおどろおどろしく 源氏物語

派生関係：

原義：正体不明の霊的な気
  ├→ ❶ 人に取り憑く霊 → 憑依する存在（最重要義）
  ├→ ❷ その正体 → 死霊・生霊
  └→ ❸ その結果 → 祟り、怪異現象
      → 現代語:「もののけ」＝妖怪・化け物（広義化）
現代語との決定的な違い

現代語「もののけ」＝ 妖怪、化け物、お化け（広義） 古語「もののけ」＝ 人に取り憑く死霊・生霊（具体的な憑依現象） 古文の「もののけ」は特定の人物に取り憑いて病を引き起こす。アニメの妖怪とは違う。

判別基準と文脈解説

意味 どんな文脈で？ 共起しやすい語
❶憑依する霊 人が病に倒れ、原因が霊とされる場面。平安貴族社会では病気＝もののけの仕業。 「わづらふ」「苦しむ」「憑く」
❷死霊・生霊 霊の正体が判明する場面。死者の怨念、または生きている人の嫉妬が霊となる。 「出で来」「あらはる」「御息所」
❸祟り 怪異現象が起きる場面。物音・発光・人の急変等。 「おどろおどろし」「あやし」
判別のコツ

古文で「もののけ」は病気の原因として登場することがほとんど。 「もののけにわづらふ」＝物の怪に苦しむ＝霊に取り憑かれて病む。 源氏物語の六条御息所の生霊が最も有名な例。

用例（意味番号付き）

意味 用例 文脈解説
❶憑依する霊 「もののけにわづらひたまふ」（源氏物語） 物の怪に苦しみなさる→霊の憑依による病
❷生霊 「六条御息所の御もののけ」（源氏物語） 六条御息所の生霊→生きている人の嫉妬が霊に
❸祟り 「もののけなどおどろおどろしく」（源氏物語） 物の怪などが恐ろしく→怪異現象の描写

入試での注意点

最重要

  • 源氏物語の六条御息所の生霊は必出知識
  • 生きている人の嫉妬・怨念が「もののけ」となる
  • 平安時代の病気観：病＝もののけの仕業 → 加持祈祷で治療
  • 「もののけ」が出たら病気・憑依・祟りの文脈を想定する
  • 現代語の「妖怪」のイメージで読むと誤読する
